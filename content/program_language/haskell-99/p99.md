## 数据表示

`Left False` 是黑格，`Left True` 是待填格，`Right c` 是预填字符。连续两个以上可填格形成横向或纵向单词槽。

```haskell
import Data.List (delete, minimumBy, transpose)
import Data.Ord (comparing)

data Crossword = Crossword
  { word :: [String]
  , grid :: [[Either Bool Char]]
  } deriving (Eq, Show)

type Position = (Int, Int)
type Site = [Position]
type Board = [[Maybe Char]]
```

## 提取单词槽

```haskell
allSites :: [[Either Bool Char]] -> [Site]
allSites puzzle = horizontal ++ vertical
  where
    rows = length puzzle
    cols = maximum (0 : map length puzzle)
    cell r c
      | r < rows && c < length (puzzle !! r) = puzzle !! r !! c
      | otherwise = Left False
    spot r c = case cell r c of
      Left False -> Nothing
      _          -> Just (r,c)
    horizontal = concat
      [runsFrom [spot r c | c <- [0..cols-1]] | r <- [0..rows-1]]
    vertical = concat
      [runsFrom [spot r c | r <- [0..rows-1]] | c <- [0..cols-1]]

runsFrom :: [Maybe Position] -> [Site]
runsFrom = filter ((>= 2) . length) . finish . foldl step ([],[])
  where
    step (done,current) Nothing = (reverse current:done, [])
    step (done,current) (Just p) = (done, p:current)
    finish (done,current) = reverse (reverse current:done)
```

## 回溯求解核心

### 方法一：每轮重新计算候选词

```haskell
solveCrossword :: Crossword -> Maybe Board
solveCrossword (Crossword words puzzle) =
  search initial (allSites puzzle) words
  where
    initial = [[initialCell cell | cell <- row] | row <- puzzle]
    initialCell (Right c) = Just c
    initialCell _         = Nothing

search board [] _ = Just board
search board remaining words = firstJust
  [search (place board site candidate) (delete site remaining) (delete candidate words)
  | candidate <- candidates]
  where
    ranked = [(site, matching board site words) | site <- remaining]
    (site,candidates) = minimumBy (comparing (length . snd)) ranked

matching board site words =
  [w | w <- words, length w == length site, fits board site w]

fits board site text = and
  [maybe True (== c) (board !! r !! col)
  | ((r,col),c) <- zip site text]

place board site text = foldl put board (zip site text)
  where
    put rows ((r,c),value) =
      take r rows ++ [take c row ++ [Just value] ++ drop (c+1) row]
                  ++ drop (r+1) rows
      where row = rows !! r

firstJust [] = Nothing
firstJust (Just value:_) = Just value
firstJust (Nothing:rest) = firstJust rest
```

搜索每次选择候选词最少的槽（MRV），填词后交叉位置会自动限制其他槽。

### 方法二：前向检查候选域

```haskell
solveCrosswordForward :: Crossword -> Maybe Board
solveCrosswordForward (Crossword words puzzle) =
  searchForward initial domains words
  where
    initial = [[initialCell cell | cell <- row] | row <- puzzle]
    initialCell (Right c) = Just c
    initialCell _         = Nothing
    domains =
      [(site, matching initial site words) | site <- allSites puzzle]

searchForward :: Board -> [(Site, [String])] -> [String] -> Maybe Board
searchForward board [] _ = Just board
searchForward board domains words = firstJust
  [advance candidate | candidate <- candidates]
  where
    (site, candidates) = minimumBy (comparing (length . snd)) domains

    advance candidate
      | any (null . snd) updatedDomains = Nothing
      | otherwise = searchForward board' updatedDomains words'
      where
        board' = place board site candidate
        words' = delete candidate words
        remainingSites =
          [otherSite | (otherSite, _) <- domains, otherSite /= site]
        updatedDomains =
          [(otherSite, matching board' otherSite words')
          | otherSite <- remainingSites]
```

方法一只在选中某个槽时计算它的候选词；前向检查在每次填词后更新所有剩余槽的候选域。任一候选域变空就立即回溯，避免等到该槽被 MRV 选中才发现冲突。

## 方法对比

| 方法 | 填词后的工作 |
|------|--------------|
| 重新计算 | 下一轮只计算当前 MRV 排名所需候选 |
| 前向检查 | 更新所有剩余候选域并检测空域 |

## 测试

```haskell
>>> fmap (map (map (maybe '#' id))) (solveCrossword crosswordPuzzle)
Just ["##P##","##O##","ALPHA","##P#R","##Y#E","####S"]
>>> solveCrosswordForward crosswordPuzzle == solveCrossword crosswordPuzzle
True
```

大型填字题的搜索空间仍可能很大，但“按长度预筛 + 交叉字符传播 + MRV”构成了完整的约束求解主线。
