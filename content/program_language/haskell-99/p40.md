# P40 - 哥德巴赫猜想

> Find two primes that sum to a given even integer
>
> 官方模块：`Problems.P40`　核心函数：`goldbach`

## 题目描述

哥德巴赫猜想认为每个大于 2 的偶数都能写成两个素数之和。给定这样的偶数，找出其中一组分解。

## 函数签名与实现

```haskell
goldbach :: Integral a => a -> (a, a)
goldbach n
  | n <= 2 || odd n = error "goldbach: even input greater than 2 required"
  | otherwise = head
      [(p, n - p) | p <- takeWhile (<= n `div` 2) primes
                  , isPrime (n - p)]
```

只枚举 $p\le n/2$ 可以避免把 `(p,q)` 与 `(q,p)` 重复考虑。这里复用 P31 和 P39。

## 测试

```haskell
>>> goldbach 12
(5,7)
>>> goldbach 28
(5,23)
>>> let (p,q) = goldbach 100 in p + q == 100 && isPrime p && isPrime q
True
```

> [!WARNING] 定义域
> `head` 依赖哥德巴赫猜想在输入上成立。函数明确把定义域限制为大于 2 的偶数；它不是对任意整数都安全的查询接口。
