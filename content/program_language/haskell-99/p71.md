# P71 - 多路树的内部路径长度

> Internal path length of a tree
>
> 官方模块：`Problems.P71`　核心函数：`internalPathLength`

---

## 题目描述

内部路径长度定义为根到每个节点的路径长度之和。根深度为 0，根的孩子深度为 1。

## 实现

### 方法一：深度优先递归

```haskell
internalPathLength :: MultiwayTree a -> Int
internalPathLength = go 0
  where
    go depth (MultiwayTree _ children) =
      depth + sum (map (go (depth + 1)) children)
```

深度作为只读上下文向下传递，当前节点贡献自己的深度，再汇总全部孩子。

### 方法二：先根遍历（等价）

internalPathLength :: MultiwayTree a -> Int
internalPathLength = go 0
  where
    go depth (MultiwayTree x children) =
      depth + sum (map (go (depth + 1)) children)

与标准解相同，已是最优。

## 测试

```haskell
>>> internalPathLength multitree1
0
>>> internalPathLength multitree5
9
>>> internalPathLength (MultiwayTree 'a' [MultiwayTree 'b' []])
1
```

每个节点访问一次，时间复杂度 $O(n)$，递归栈深度为树高 $O(h)$。
