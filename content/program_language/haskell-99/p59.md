# P59 - 构造高度平衡二叉树

> Construct height-balanced binary trees
>
> 官方模块：`Problems.P59`　核心函数：`heightBalancedTrees`

## 题目描述

高度平衡要求每个节点的左右子树高度差不超过 1。给定高度 $h$，生成所有恰好为该高度的树形。

## 实现

```haskell
heightBalancedTrees :: Int -> [Tree ()]
heightBalancedTrees 0 = [Empty]
heightBalancedTrees 1 = [leaf ()]
heightBalancedTrees h
  | h < 0     = []
  | otherwise =
      combine tall tall ++ combine tall short ++ combine short tall
  where
    tall  = heightBalancedTrees (h - 1)
    short = heightBalancedTrees (h - 2)
    combine ls rs = [Branch () l r | l <- ls, r <- rs]
```

要让根的高度为 $h$，至少一侧必须高 $h-1$；另一侧只能是 $h-1$ 或 $h-2$，所以恰好有三种高度组合。

## 测试

```haskell
>>> length (heightBalancedTrees 3)
15
>>> all ((== 3) . treeHeight) (heightBalancedTrees 3)
True
>>> heightBalancedTrees 0
[Empty]
```
