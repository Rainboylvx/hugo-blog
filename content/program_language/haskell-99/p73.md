# P73 - 多路树的 S 表达式表示

> Multiway tree representation with S-expressions

## 编码规则

叶子直接写字符；有孩子的节点写成 `(根 子树1 子树2 ...)`。

### 方法一：递归下降

```haskell
treeToSexp :: MultiwayTree Char -> String
treeToSexp (MultiwayTree x []) = [x]
treeToSexp (MultiwayTree x children) =
  "(" ++ [x] ++ " " ++ unwords (map treeToSexp children) ++ ")"
```

## 递归下降解析

```haskell
sexpToTree :: String -> Maybe (MultiwayTree Char)
sexpToTree input = do
  (tree, rest) <- parse (dropWhile (== ' ') input)
  if null (dropWhile (== ' ') rest) then Just tree else Nothing

parse :: String -> Maybe (MultiwayTree Char, String)
parse [] = Nothing
parse ('(':rest) = do
  (root, afterRoot) <- nextChar rest
  (children, remaining) <- parseList afterRoot
  pure (MultiwayTree root children, remaining)
parse (')':_) = Nothing
parse (x:rest) = Just (MultiwayTree x [], rest)

nextChar input = case dropWhile (== ' ') input of
  x:rest | x /= '(' && x /= ')' -> Just (x, rest)
  _ -> Nothing

parseList input = case dropWhile (== ' ') input of
  ')':rest -> Just ([], rest)
  []       -> Nothing
  xs       -> do
    (child, afterChild) <- parse xs
    (siblings, rest)    <- parseList afterChild
    pure (child : siblings, rest)
```

### 方法二：`ReadP` 解析组合子

```haskell
import Text.ParserCombinators.ReadP

sexpToTreeReadP :: String -> Maybe (MultiwayTree Char)
sexpToTreeReadP input = case
  [tree | (tree, "") <- readP_to_S (skipSpaces *> treeP <* skipSpaces <* eof) input] of
    []    -> Nothing
    trees -> Just (last trees)
  where
    treeP = branchP <++ leafP

    leafP = MultiwayTree <$> nodeChar <*> pure []

    branchP = do
      char '('
      skipSpaces
      root <- nodeChar
      children <- many (skipSpaces *> treeP)
      skipSpaces
      char ')'
      pure (MultiwayTree root children)

    nodeChar = satisfy (\c -> c /= '(' && c /= ')' && c /= ' ')

treeToSexpDL :: MultiwayTree Char -> String
treeToSexpDL tree = build tree ""
  where
    build (MultiwayTree value []) = (value :)
    build (MultiwayTree value children) =
      ('(' :) . (value :) . foldr addChild id children . (')' :)
    addChild child rest = (' ' :) . build child . rest
```

`ReadP` 用组合子直接描述“叶子或括号分支”的文法；差异列表编码避免多层 `++`。它与方法一显式管理剩余字符串的方式形成对照。

## 方法对比

| 方法 | 空白和分支处理 |
|------|----------------|
| 手写递归下降 | 显式 `dropWhile` 和剩余字符串 |
| `ReadP` | `skipSpaces`、`many` 和选择组合子 |

## 测试

```haskell
>>> treeToSexp multitree5
"(a (f g) c (b d e))"
>>> sexpToTree (treeToSexp multitree5) == Just multitree5
True
>>> sexpToTree "(a b"
Nothing
>>> sexpToTreeReadP (treeToSexp multitree5) == Just multitree5
True
>>> treeToSexpDL multitree5 == treeToSexp multitree5
True
```
