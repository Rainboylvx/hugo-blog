# P21 - 在指定位置插入元素

> Insert an element at a given position into a list
>
> 官方模块：`Problems.P21`　核心函数：`insertAt`

## 题目描述

把元素插入列表的第 $n$ 个位置。题目使用从 1 开始的位置编号，因此位置 1 表示插到列表最前面。

## 函数签名

```haskell
insertAt :: a -> [a] -> Int -> [a]
```

## 思路

在位置 $n$ 插入，等价于在原列表前 $n-1$ 个元素之后插入。用 `splitAt` 一次得到前缀和后缀，再把新元素接在二者之间。

## 实现

```haskell
insertAt :: a -> [a] -> Int -> [a]
insertAt x xs n = front ++ x : back
  where
    (front, back) = splitAt (max 0 (n - 1)) xs
```

`++` 的右侧是 `x : back`，所以这里只复制前缀；若位置超过列表长度，`splitAt` 会让 `back` 为空，新元素自然追加到末尾。

## 测试

```haskell
>>> insertAt 'X' "abcd" 2
"aXbcd"
>>> insertAt 0 [1,2,3] 1
[0,1,2,3]
>>> insertAt 4 [1,2,3] 99
[1,2,3,4]
```

## 要点总结

- 题目位置从 1 开始，传给 `splitAt` 时要减 1。
- `splitAt` 比手写下标递归更直接，也清楚定义了越界行为。
