# P52 - 转换为合取范式

> Convert a formula to conjunctive normal form
>
> 官方模块：`Problems.P52`　核心函数：`toConjunctiveNormalForm`

---

## 数据类型

```haskell
import Data.List (nub, sort)

data Formula
  = Value Bool | Variable String | Complement Formula
  | Disjoin [Formula] | Conjoin [Formula]
  deriving (Eq, Ord, Show)
```

合取范式（CNF）是“若干析取子句的合取”。这里用真值表构造规范 CNF：对每个使原式为假的赋值，生成一个只排除该赋值的子句。

## 实现

```haskell
toConjunctiveNormalForm :: Formula -> Formula
toConjunctiveNormalForm formula = Conjoin
  [ Disjoin [literal name value | (name,value) <- env]
  | env <- environments (variables formula)
  , not (evaluate env formula)
  ]
  where
    literal name True  = Complement (Variable name)
    literal name False = Variable name

variables :: Formula -> [String]
variables (Value _)      = []
variables (Variable x)   = [x]
variables (Complement f) = variables f
variables (Disjoin fs)   = sort . nub $ concatMap variables fs
variables (Conjoin fs)   = sort . nub $ concatMap variables fs

environments :: [String] -> [[(String, Bool)]]
environments [] = [[]]
environments (x:xs) =
  [(x,value) : env | value <- [False,True], env <- environments xs]

evaluate :: [(String, Bool)] -> Formula -> Bool
evaluate _   (Value b)      = b
evaluate env (Variable x)   = maybe False id (lookup x env)
evaluate env (Complement f) = not (evaluate env f)
evaluate env (Disjoin fs)   = any (evaluate env) fs
evaluate env (Conjoin fs)   = all (evaluate env) fs
```

## 测试

```haskell
>>> let p = Variable "p"; q = Variable "q"
>>> toConjunctiveNormalForm (Disjoin [p, Conjoin [q, Complement p]])
Conjoin [Disjoin [Variable "p",Variable "q"]]
>>> toConjunctiveNormalForm (Disjoin [p, Complement p])
Conjoin []
```

空合取表示真。规范 CNF 最坏有 $2^n$ 个子句，优点是实现短且语义容易验证。
