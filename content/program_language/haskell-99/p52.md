# P52 - 合取范式

> Conjunctive normal form
>
> 官方模块：`Problems.P52`　核心函数：`toConjunctiveNormalForm`

## 题目描述

将命题逻辑公式转换为合取范式（CNF）。CNF 是子句的合取，每个子句是文字的析取。

## 数据类型

```haskell
import Data.List (nub, sort)

data Formula
  = Value Bool | Variable String | Complement Formula
  | Disjoin [Formula] | Conjoin [Formula]
  deriving (Eq, Ord, Show)
```

## 函数签名

```haskell
toConjunctiveNormalForm :: Formula -> Formula
```

## 实现

### 方法一：真值表法（枚举所有 falsifying 行）

```haskell
toConjunctiveNormalForm :: Formula -> Formula
toConjunctiveNormalForm formula = Conjoin
  [ Disjoin [literal name value | (name,value) <- env]
  | env <- environments (variables formula)
  , not (evaluate env formula)
  ]
  where
    literal name True  = Complement (Variable name)
    literal name False = Variable name

variables :: Formula -> [String]
variables (Value _)      = []
variables (Variable x)   = [x]
variables (Complement f) = variables f
variables (Disjoin fs)   = sort . nub $ concatMap variables fs
variables (Conjoin fs)   = sort . nub $ concatMap variables fs

environments :: [String] -> [[(String, Bool)]]
environments [] = [[]]
environments (x:xs) =
  [(x,value) : env | value <- [False,True], env <- environments xs]

evaluate :: [(String, Bool)] -> Formula -> Bool
evaluate _   (Value b)      = b
evaluate env (Variable x)   = maybe False id (lookup x env)
evaluate env (Complement f) = not (evaluate env f)
evaluate env (Disjoin fs)   = any (evaluate env) fs
evaluate env (Conjoin fs)   = all (evaluate env) fs
```

对于每个使公式为假的环境，生成一个对应子句排除它。

### 方法二：代数变换法

```haskell
toConjunctiveNormalForm :: Formula -> Formula
toConjunctiveNormalForm = fromClauses . cnfClauses . toNNF
  where
    fromClauses clauses = Conjoin (map Disjoin clauses)

    toNNF (Complement (Conjoin fs)) = Disjoin (map (toNNF . Complement) fs)
    toNNF (Complement (Disjoin fs)) = Conjoin (map (toNNF . Complement) fs)
    toNNF (Complement (Complement f)) = toNNF f
    toNNF (Complement (Value b)) = Value (not b)
    toNNF (Conjoin fs) = Conjoin (map toNNF fs)
    toNNF (Disjoin fs) = Disjoin (map toNNF fs)
    toNNF f = f

    cnfClauses (Value True)  = []
    cnfClauses (Value False) = [[]]
    cnfClauses (Conjoin fs)  = concatMap cnfClauses fs
    cnfClauses (Disjoin fs)  = foldl distribute [[]] (map cnfClauses fs)
    cnfClauses literal       = [[literal]]

    distribute left right =
      [nub (leftClause ++ rightClause)
      | leftClause <- left, rightClause <- right]
```

`toNNF` 先用德摩根律把否定推到变量上。`cnfClauses` 用 `[[Formula]]` 表示“子句的合取”，遇到析取时计算两个子句集合的笛卡尔积，从而完整应用分配律。这个方法不枚举真值环境，通常比方法一生成更紧凑的 CNF。

## 方法对比

| 方法 | 特点 | 代价 |
|------|------|------|
| 真值表 | 定义直接，结果容易验证 | 变量数为 n 时枚举 2^n 个环境 |
| 代数变换 | 不枚举赋值，保留公式结构 | 分配律仍可能导致指数膨胀 |

## 测试

```haskell
>>> toConjunctiveNormalForm (Disjoin [Variable "x", Variable "y"])
Conjoin [Disjoin [Variable "x",Variable "y"]]
>>> toConjunctiveNormalForm
...   (Disjoin [Variable "x", Conjoin [Variable "y", Variable "z"]])
Conjoin [Disjoin [Variable "x",Variable "y"],Disjoin [Variable "x",Variable "z"]]
```
