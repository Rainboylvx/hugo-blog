# P33 - 判断互质

> Determine whether two positive integers are coprime
>
> 官方模块：`Problems.P33`　核心函数：`coprime`

---

[← P32 最大公约数](./p32) | [P34 欧拉函数 →](./p34)

---

## 题目描述

如果两个整数的最大公约数为 1，就称它们互质。互质不要求两个数本身都是素数，例如 35 和 64 互质。

## 函数签名

```haskell
coprime :: Integral a => a -> a -> Bool
```

## 实现

### 方法一：基于 GCD（最直接）

```haskell
coprime :: Integral a => a -> a -> Bool
coprime a b = myGCD a b == 1
```

P32 已经完成全部计算，这道题的重点是把数学定义直接组合成程序。

### 方法二：不用 GCD，遍历检查公因数

```haskell
coprime :: Integral a => a -> a -> Bool
coprime a b = null [d | d <- [2..min (abs a) (abs b)], a `mod` d == 0 && b `mod` d == 0]
```

直接从定义出发：不存在大于 1 的公因数。效率低但逻辑显式。

### 方法三：只枚举到平方根

```haskell
coprime :: Integral a => a -> a -> Bool
coprime a b = all noCommonFactor [1..integerSqrt smaller]
  where
    smaller = min (abs a) (abs b)
    noCommonFactor divisor
      | smaller `mod` divisor /= 0 = True
      | otherwise =
          not (nonTrivial divisor && abs b `mod` divisor == 0)
          && not (nonTrivial paired && abs b `mod` paired == 0)
      where
        paired = smaller `div` divisor
    nonTrivial factor = factor > 1

integerSqrt :: Integral a => a -> a
integerSqrt n = last (takeWhile (\x -> x * x <= n) [0..])
```

只枚举到平方根时，必须同时检查成对因子 `smaller div divisor`。例如 7 和 14 的公共因子 7 大于 $\sqrt 7$，但它会作为因子 1 的配对项被检查到。

## 方法对比

| 方法 | 特点 | 效率 |
|------|------|------|
| GCD 法 | 一句话，最简洁 | O(log min(a,b)) |
| 枚举公因数 | 从定义出发，教学直观 | O(min(a,b)) |
| 枚举到 √n | 不依赖 GCD，比方法二快 | O(√min(a,b)) |

## 测试

```haskell
>>> coprime 35 64
True
>>> coprime 21 14
False
>>> coprime 1 100
True
>>> coprime 7 14
False
```
