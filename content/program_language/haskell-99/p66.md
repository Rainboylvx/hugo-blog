# P66 - 二叉树紧凑布局

> Binary tree layout: compact
>
> 官方模块：`Problems.P66`　核心函数：`layoutCompact`

## 题目描述

尽可能紧凑地布局二叉树，使得宽度最小。同一层的节点不重叠，且父节点居中于子节点。

## 函数签名

```haskell
layoutCompact :: Tree a -> Tree (a, (Int, Int))
```

## 实现

### 方法一：相对坐标 + 绝对定位

```haskell
type Bounds = [(Int, Int)]
data Relative a = REmpty | RNode a Int Bounds (Relative a) (Relative a)

layoutCompact :: Tree a -> Tree (a, (Int, Int))
layoutCompact tree = absolute relative (1 - leftMost relative) 1
  where
    relative = makeRelative tree 1

    makeRelative Empty _ = REmpty
    makeRelative (Branch x l r) depth =
      let lRel = makeRelative l (depth + 1)
          rRel = makeRelative r (depth + 1)
          lBounds = extBounds lRel
          rBounds = extBounds rRel
          myX = 1 + maxWidth lBounds 1
          myBounds = (myX, myX) : merge (shiftBounds (-myX) lBounds) (shiftBounds myX rBounds)
      in RNode x myX myBounds lRel rRel

    absolute REmpty _ _ = Empty
    absolute (RNode x xRel _ l r) offset y =
      Branch (x, (xRel + offset, y))
             (absolute l (offset + xRel) (y + 1))
             (absolute r (offset + xRel) (y + 1))

    leftMost (RNode _ x _ _ _) = x
    leftMost REmpty = 0

    extBounds REmpty = [(0, 0)]
    extBounds (RNode _ _ bs _ _) = bs

    shiftBounds shift = map (\(a, b) -> (a + shift, b + shift))

    maxWidth bounds level = maximum (0 : [max a b | (a, b) <- drop (level - 1) bounds])

    merge [] bs = bs
    merge as [] = as
    merge (a:as) (b:bs) = (min (fst a) (fst b), max (snd a) (snd b)) : merge as bs
```
