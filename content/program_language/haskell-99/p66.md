# P66 - 二叉树紧凑布局

> Binary tree layout: compact
>
> 官方模块：`Problems.P66`　核心函数：`layoutCompact`

## 题目描述

尽可能紧凑地布局二叉树，使得宽度最小。同一层的节点不重叠，且父节点居中于子节点。

## 函数签名

```haskell
layoutCompact :: Tree a -> Tree (a, (Int, Int))
```

## 实现

### 方法一：相对坐标 + 绝对定位

```haskell
type Bounds = [(Int, Int)]
data Relative a = REmpty | RNode a Int Bounds (Relative a) (Relative a)

layoutCompact :: Tree a -> Tree (a, (Int, Int))
layoutCompact tree = absolute relative (1 - leftMost relative) 1
  where
    relative = compact tree

compact :: Tree a -> Relative a
compact Empty = REmpty
compact (Branch value left right) =
  RNode value 0 bounds (move (-distance) l) (move distance r)
  where
    l = compact left
    r = compact right
    distance = safeDistance (contour l) (contour r)
    bounds = merge (shiftBounds (-distance) (contour l))
                   (shiftBounds distance (contour r))

contour :: Relative a -> Bounds
contour REmpty = []
contour (RNode _ _ bounds _ _) = (0, 0) : bounds

move :: Int -> Relative a -> Relative a
move _ REmpty = REmpty
move distance (RNode value position bounds left right) =
  RNode value (position + distance) bounds left right

shiftBounds :: Int -> Bounds -> Bounds
shiftBounds distance =
  map (\(left, right) -> (left + distance, right + distance))

merge :: Bounds -> Bounds -> Bounds
merge [] ys = ys
merge xs [] = xs
merge ((left, _):xs) ((_, right):ys) =
  (left, right) : merge xs ys

safeDistance :: Bounds -> Bounds -> Int
safeDistance left right = head
  [distance | distance <- [1..], and (zipWith (separated distance) left right)]
  where
    separated distance (_, leftRight) (rightLeft, _) =
      leftRight - distance < rightLeft + distance

leftMost :: Relative a -> Int
leftMost REmpty = 0
leftMost (RNode _ position bounds _ _) =
  minimum (position : map fst bounds)

absolute :: Relative a -> Int -> Int -> Tree (a, (Int, Int))
absolute REmpty _ _ = Empty
absolute (RNode value offset _ left right) parentX depth =
  Branch (value, (x, depth))
    (absolute left x (depth + 1))
    (absolute right x (depth + 1))
  where
    x = parentX + offset
```

每棵相对布局保存逐层的左右轮廓。合并子树时寻找最小安全距离，最后一次遍历把相对坐标转换成绝对坐标。

### 方法二：直接检查逐层坐标

```haskell
layoutCompactByLevels :: Tree a -> Tree (a, (Int, Int))
layoutCompactByLevels tree = normalize (build tree)
  where
    build Empty = Empty
    build (Branch value left right) =
      Branch (value, (0, 1))
        (shiftLayout (-distance) 1 leftLayout)
        (shiftLayout distance 1 rightLayout)
      where
        leftLayout = build left
        rightLayout = build right
        distance = head
          [d | d <- [1..], levelsSeparated d leftLayout rightLayout]

    normalize Empty = Empty
    normalize layout =
      shiftLayout (1 - minimum (map fst (positions layout))) 0 layout

levelsSeparated :: Int -> Tree (a, (Int, Int)) -> Tree (a, (Int, Int)) -> Bool
levelsSeparated distance left right = and
  [maximum leftLevel - distance < minimum rightLevel + distance
  | (leftLevel, rightLevel) <- zip (levels left) (levels right)]

levels :: Tree (a, (Int, Int)) -> [[Int]]
levels tree = takeWhile (not . null) [atDepth depth tree | depth <- [1..]]
  where
    atDepth _ Empty = []
    atDepth target (Branch (_, (x, y)) left right)
      | target == y = [x]
      | otherwise = atDepth target left ++ atDepth target right

positions :: Tree (a, (Int, Int)) -> [(Int, Int)]
positions Empty = []
positions (Branch (_, position) left right) =
  position : positions left ++ positions right

shiftLayout :: Int -> Int -> Tree (a, (Int, Int)) -> Tree (a, (Int, Int))
shiftLayout _ _ Empty = Empty
shiftLayout dx dy (Branch (value, (x, y)) left right) =
  Branch (value, (x + dx, y + dy))
    (shiftLayout dx dy left)
    (shiftLayout dx dy right)
```

这个版本不建立专门的轮廓类型，而是先生成带相对坐标的子树，再逐层读取实际横坐标并测试安全距离。代码更直观，但重复扫描子树，效率低于方法一。

## 方法对比

| 方法 | 中间数据 | 特点 |
|------|----------|------|
| 相对坐标 + 轮廓 | 每层左右边界 | 合并高效，推荐 |
| 直接检查逐层坐标 | 完整坐标树 | 容易理解，重复遍历较多 |

## 测试

```haskell
>>> layoutCompact (Branch 'a' (Branch 'b' Empty (leaf 'c')) (leaf 'd'))
Branch ('a',(2,1)) (Branch ('b',(1,2)) Empty (Branch ('c',(2,3)) Empty Empty)) (Branch ('d',(3,2)) Empty Empty)
>>> positions (layoutCompactByLevels (Branch 'a' (leaf 'b') (leaf 'c')))
[(2,1),(1,2),(3,2)]
```
