# P66 - 二叉树紧凑布局

> Compact binary tree layout
>
> 官方模块：`Problems.P66`　核心函数：`layoutCompact`

## 核心思想

每棵子树记录各深度的 `(最左位置, 最右位置)` 轮廓。合并左右子树时，寻找最小距离 $d$，使左轮廓右边界平移 `-d` 后，严格位于右轮廓左边界平移 `d` 的左侧。

## 实现

```haskell
type Bounds = [(Int, Int)]
data Relative a = REmpty | RNode a Int Bounds (Relative a) (Relative a)

layoutCompact :: Tree a -> Tree (a, (Int, Int))
layoutCompact tree = absolute relative (1 - leftMost relative) 1
  where relative = compact tree

compact :: Tree a -> Relative a
compact Empty = REmpty
compact (Branch value left right) =
  RNode value 0 bounds (move (-distance) l) (move distance r)
  where
    l = compact left
    r = compact right
    distance = safeDistance (contour l) (contour r)
    bounds = merge (shiftBounds (-distance) (contour l))
                   (shiftBounds distance (contour r))

contour REmpty                = []
contour (RNode _ _ bs _ _)    = (0,0) : bs

move _ REmpty = REmpty
move d (RNode x p bs l r) = RNode x (p+d) bs l r

shiftBounds d = map (\(l,r) -> (l+d,r+d))

merge [] ys = ys
merge xs [] = xs
merge ((l,_):xs) ((_,r):ys) = (l,r) : merge xs ys

safeDistance left right = head
  [d | d <- [1..], and (zipWith (separated d) left right)]
  where separated d (_,lr) (rl,_) = lr - d < rl + d

leftMost REmpty = 0
leftMost (RNode _ p bs _ _) = minimum (p : map fst bs)

absolute REmpty _ _ = Empty
absolute (RNode value offset _ left right) parentX depth =
  Branch (value,(x,depth))
    (absolute left x (depth+1))
    (absolute right x (depth+1))
  where x = parentX + offset
```

> [!IMPORTANT] 轮廓而非节点总宽度
> 两棵子树能否靠近，取决于它们在每一层的相向边界。只比较整体宽度会错过可以交错利用的空白。

## 验证要点

- 每个子节点仍在父节点相应方向；
- 同一坐标没有两个节点；
- 任意再缩小一次左右间距都会在某层发生碰撞。
