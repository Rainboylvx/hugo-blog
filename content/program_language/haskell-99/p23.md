# P23 - 从列表随机选 N 个元素

> Extract a given number of randomly selected elements from a list
>
> 官方模块：`Problems.P23`　核心函数：`randomSelect`

---

[← P22 生成整数范围](./p22) | [P24 从 1..M 中随机选 N 个不同数 →](./p24)

---

## 题目描述

从列表中随机选择 $N$ 个不重复的元素。需要接受一个随机数生成器并返回更新后的生成器——这是 Haskell 纯函数随机数的标准做法。

## 函数签名

```haskell
import System.Random (RandomGen)

randomSelect :: RandomGen g => [a] -> Int -> g -> ([a], g)
```

`RandomGen g` 是类型类约束，表示任何随机数生成器都可以。

## 实现

### 方法一：随机移除（选中的拿走）

```haskell
import System.Random (RandomGen, randomR)

randomSelect :: RandomGen g => [a] -> Int -> g -> ([a], g)
randomSelect xs count gen = go xs (max 0 count) gen []
  where
    go [] _ g picked = (reverse picked, g)
    go _  0 g picked = (reverse picked, g)
    go pool k g picked =
      let (i, g')       = randomR (0, length pool - 1) g
          (x, pool')    = removeAt0 i pool
      in go pool' (k - 1) g' (x : picked)

removeAt0 :: Int -> [a] -> (a, [a])
removeAt0 i xs =
  case splitAt i xs of
    (front, x : back) -> (x, front ++ back)
    _ -> error "removeAt0: index out of bounds"
```

核心思路：每次从剩余池中随机选一个元素，将其从池中移除并加入结果列表。

> 注意：`randomR (0, length pool - 1)` 在空列表时会除零。好在 `pool` 为空时，前面的 `go []` 模式已经截住了。

### 方法二：蓄水池抽样

```haskell
import System.Random (RandomGen, randomR)

randomSelect :: RandomGen g => [a] -> Int -> g -> ([a], g)
randomSelect xs count gen
  | count <= 0 = ([], gen)
  | otherwise  = go size reservoir rest gen
  where
    size = min count (length xs)
    (reservoir, rest) = splitAt size xs

    go _ picked [] g = (picked, g)
    go seen picked (x:remaining) g =
      let (slot, g') = randomR (0, seen) g
          picked'
            | slot < size = replaceAt slot x picked
            | otherwise   = picked
      in go (seen + 1) picked' remaining g'

replaceAt :: Int -> a -> [a] -> [a]
replaceAt i x xs = front ++ x : drop 1 back
  where
    (front, back) = splitAt i xs
```

先把前 $N$ 个元素放入蓄水池。读到第 $i$ 个后续元素时，在 $0$ 到 $i$ 之间随机选择位置；只有位置落在蓄水池内才执行替换。这样只需扫描一次列表，并且每个元素最终被选中的概率相同。

### 方法三：基于洗牌的固定抽样

```haskell
import System.Random (RandomGen, randomR)

randomSelect :: RandomGen g => [a] -> Int -> g -> ([a], g)
randomSelect xs k gen = (take k shuffled, gen'')
  where
    (shuffled, gen'') = fisherYates xs gen

fisherYates :: RandomGen g => [a] -> g -> ([a], g)
fisherYates [] gen = ([], gen)
fisherYates xs gen = go (length xs) xs gen
  where
    go 0 acc gen = (acc, gen)
    go n acc gen =
      let (i, gen') = randomR (0, n - 1) gen
          swapped   = swap i (n - 1) acc
      in go (n - 1) swapped gen'

swap :: Int -> Int -> [a] -> [a]
swap i j xs
  | i == j    = xs
  | otherwise =
      [ if k == i then xj else if k == j then xi else x
      | (k, x) <- zip [0..] xs
      ]
  where
    xi = xs !! i
    xj = xs !! j
```

先对整个列表做 Fisher-Yates 洗牌，再取前 K 个。适合 K 接近列表长度的场景。

## 方法对比

| 方法 | 特点 | 效率 |
|------|------|------|
| 随机移除 | 每轮从候选池删除一个元素 | O(N × len xs) |
| 蓄水池抽样 | 单次扫描，只保留 N 个元素 | O(len xs × N)，数组实现可到 O(len xs) |
| 先洗牌再取 | 得到完整随机排列后截取 | 当前列表实现 O(len xs²) |

## 测试

```haskell
>>> import System.Random (mkStdGen)
>>> let (picked, _) = randomSelect "abcdefgh" 3 (mkStdGen 111)
>>> length picked
3
>>> let (picked, _) = randomSelect [1..5] 10 (mkStdGen 1)
>>> length picked    -- 最多只有 5 个可选
5
>>> let (picked, _) = randomSelect [1..20] 5 (mkStdGen 7)
>>> all (`elem` [1..20]) picked && length picked == 5
True
```
