# P68 - 根据中序和前序序列重建二叉树

> In-order and pre-order sequences of binary trees
>
> 官方模块：`Problems.P68`　核心函数：`inorder`、`preorder`、`ordersToTree`

## 遍历

```haskell
inorder :: Tree a -> [a]
inorder Empty = []
inorder (Branch x left right) = inorder left ++ [x] ++ inorder right

preorder :: Tree a -> [a]
preorder Empty = []
preorder (Branch x left right) = [x] ++ preorder left ++ preorder right
```

## 重建思路

前序首元素是根。在中序序列中找到根，其左侧长度就是左子树节点数；据此切分剩余前序序列，再递归重建两侧。题目约定节点值互不相同。

```haskell
ordersToTree :: Eq a => [a] -> [a] -> Maybe (Tree a)
ordersToTree [] [] = Just Empty
ordersToTree inorderValues (root:preorderValues) = do
  let (leftIn, rest) = break (== root) inorderValues
  rightIn <- case rest of
    _:xs -> Just xs
    []   -> Nothing
  let (leftPre, rightPre) = splitAt (length leftIn) preorderValues
  left  <- ordersToTree leftIn leftPre
  right <- ordersToTree rightIn rightPre
  pure (Branch root left right)
ordersToTree _ _ = Nothing
```

## 测试

```haskell
>>> inorder tree1
"dbeacgf"
>>> preorder tree1
"abdecfg"
>>> ordersToTree (inorder tree1) (preorder tree1) == Just tree1
True
>>> ordersToTree "ab" "a"
Nothing
```

列表查找和切分使最坏复杂度为 $O(n^2)$；使用值到中序位置的映射可以优化到 $O(n)$。
