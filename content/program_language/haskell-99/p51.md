# P51 - 三重复纠错码

> Error-correcting codes
>
> 官方模块：`Problems.P51`　核心函数：`errorCorrectingEncode`、`errorCorrectingDecode`

---

## 题目描述

用长度为 3 的重复码编码布尔位：`False` 变为三个 `False`，`True` 同理。解码时对每组三位多数表决，因此每组最多一个错误仍可恢复。

## 实现

```haskell
errorCorrectingEncode :: [Bool] -> [Bool]
errorCorrectingEncode = concatMap (replicate 3)

errorCorrectingDecode :: [Bool] -> [Bool]
errorCorrectingDecode [] = []
errorCorrectingDecode xs = majority chunk : errorCorrectingDecode rest
  where
    (chunk, rest) = splitAt 3 xs
    majority bits = length (filter id bits) * 2 > length bits
```

## 随机破坏辅助函数

```haskell
import Data.List (nub, sort)
import System.Random (RandomGen, randomRs)

corrupt :: RandomGen g => g -> Int -> [Bool] -> [Bool]
corrupt gen n bits = flipAt 0 positions bits
  where
    count = min (max 0 n) (length bits)
    positions = sort . take count . nub $
      randomRs (0, length bits - 1) gen
    flipAt _ _ [] = []
    flipAt i ps@(p:rest) (b:bs)
      | i == p    = not b : flipAt (i + 1) rest bs
      | otherwise = b : flipAt (i + 1) ps bs
    flipAt _ [] remaining = remaining
```

## 测试

```haskell
>>> errorCorrectingEncode [True,False]
[True,True,True,False,False,False]
>>> errorCorrectingDecode [True,False,True,False,False,True]
[True,False]
>>> errorCorrectingDecode (errorCorrectingEncode [True,False,True])
[True,False,True]
```

代价是数据膨胀三倍，收益是每个三位块能纠正一个翻转错误。
