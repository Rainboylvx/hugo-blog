# P13 - 直接实现游程编码

> Direct run-length encoding of a list
>
> 官方模块: `Problems.P13`
> 核心函数: `encodeDirect`

## 题目描述

实现游程编码，但**不借助** `pack` 或 `group` 等中间打包步骤——直接边遍历边计数，生成 `[Encoding a]`。

结果格式与 P11 相同：单个元素用 `Single`，重复元素用 `Multiple n`。

## 函数签名

```haskell
encodeDirect :: Eq a => [a] -> [Encoding a]
```

## 思路

遍历列表，用一个计数器记录当前连续重复元素的个数：
- 当前元素和下一个不同 → 结束当前组，输出 `Single` 或 `Multiple`，重置计数器
- 当前元素和下一个相同 → 计数器加 1，继续

## 实现

### 方法一：递归 + 累加器

```haskell
encodeDirect :: Eq a => [a] -> [Encoding a]
encodeDirect [] = []
encodeDirect (x:xs) = go x 1 xs
  where
    go a n []     = [toEncoding a n]
    go a n (b:bs)
      | a == b    = go a (n + 1) bs
      | otherwise = toEncoding a n : go b 1 bs

    toEncoding x 1 = Single x
    toEncoding x n = Multiple n x
```

`go a n` 维护当前元素 `a` 和出现次数 `n`。遇到不同元素时，把当前组输出为 `Encoding`，重置计数器处理新元素。

### 方法二：使用 `span` 简化

```haskell
encodeDirect :: Eq a => [a] -> [Encoding a]
encodeDirect [] = []
encodeDirect (x:xs) = case span (== x) xs of
  (same, rest) -> let n = 1 + length same
                  in (if n == 1 then Single x else Multiple n x) : encodeDirect rest
```

`span` 自动切分出与 `x` 相等的连续段，比手动维护计数器更简洁。

### 方法三：使用 `foldr`

```haskell
encodeDirect :: Eq a => [a] -> [Encoding a]
encodeDirect = foldr go []
  where
    go x [] = [Single x]
    go x (Single y : rest)
      | x == y    = Multiple 2 x : rest
      | otherwise = Single x : Single y : rest
    go x (Multiple n y : rest)
      | x == y    = Multiple (n + 1) y : rest
      | otherwise = Single x : Multiple n y : rest
```

从右向左扫描，遇到相同元素就递增计数器，不同就插入新的 `Single`。

## 三种方法对比

| 方法 | 思路 | 特点 |
|------|------|------|
| 递归 + go | 手动维护计数器 | 逻辑显式，适合理解 |
| `span` 版 | 自动切分连续段 | 最简洁 |
| `foldr` 版 | 从右向左构建 | 巧妙但难读 |

## 测试

```haskell
>>> encodeDirect "aaaabccaadeeee"
[Multiple 4 'a', Single 'b', Multiple 2 'c', Multiple 2 'a', Single 'd', Multiple 4 'e']

>>> encodeDirect []
[]

>>> encodeDirect "a"
[Single 'a']
```
