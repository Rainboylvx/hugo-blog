# P63 - 构造并判断完全二叉树

> Construct and recognize complete binary trees
>
> 官方模块：`Problems.P63`　核心函数：`completeBinaryTree`, `isCompleteBinaryTree`

---

[← P62 收集指定层节点](./p62) | [P64 中序布局 →](./p64)

## 题目描述

完全二叉树中，除最后一层外每层都被填满，且最后一层的节点靠左排列。

## 函数签名

```haskell
completeBinaryTree :: Int -> Tree ()
isCompleteBinaryTree :: Tree a -> Bool
```

## 数组下标模型

把根编号为 1，则编号 $i$ 的左右孩子分别是 $2i$ 和 $2i+1$。完全二叉树恰好占用连续编号 $1..n$。

## 实现

### 方法一：递归下标构造 + 验证

```haskell
completeBinaryTree :: Int -> Tree ()
completeBinaryTree size = build 1
  where
    build i
      | i > size  = Empty
      | otherwise = Branch () (build (2*i)) (build (2*i + 1))

isCompleteBinaryTree :: Tree a -> Bool
isCompleteBinaryTree tree = valid tree 1
  where
    size = treeSize tree
    valid Empty i = i > size
    valid (Branch _ left right) i =
      i <= size && valid left (2*i) && valid right (2*i + 1)
```

### 方法二：isComplete 用 BFS

```haskell
isCompleteBinaryTree :: Tree a -> Bool
isCompleteBinaryTree tree = null (dropWhile notNull (bfs [tree]))
  where
    notNull Empty = True
    notNull _     = False
    bfs [] = []
    bfs (Empty : rest) = Empty : bfs rest
    bfs (Branch _ l r : rest) = l : r : bfs rest
```

BFS 层序遍历时，一旦遇到 `Empty`，之后就不能再有非空节点。

### 方法三：completeBinaryTree 用迭代构造

```haskell
import Data.Sequence (Seq, (|>), fromList, index, update)

completeBinaryTree :: Int -> Tree ()
completeBinaryTree n = build (Seq.replicate n Empty)
  -- ...
```

（可用 `Data.Sequence` 模拟数组构造。）

## 测试

```haskell
>>> treeSize (completeBinaryTree 6)
6
>>> isCompleteBinaryTree (completeBinaryTree 6)
True
>>> isCompleteBinaryTree (Branch 1 Empty (leaf 2))
False
```
