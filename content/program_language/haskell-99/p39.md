# P39 - 无限素数列表与区间素数

> Construct prime numbers in a range
>
> 官方模块：`Problems.P39`　核心函数：`primes`、`primesR`

## 题目描述

构造所有素数组成的无限列表，并据此返回闭区间 `[lo, hi]` 内的素数。

## 函数签名

```haskell
primes  :: Integral a => [a]
primesR :: Integral a => a -> a -> [a]
```

## 实现

判断候选数时，只试除已经生成且不超过平方根的素数。Haskell 的惰性求值允许 `primes` 引用自身较早的部分。

```haskell
primes :: Integral a => [a]
primes = 2 : filter primeWithKnown [3,5..]
  where
    primeWithKnown n =
      all (\p -> n `mod` p /= 0)
          (takeWhile (\p -> p * p <= n) primes)

primesR :: Integral a => a -> a -> [a]
primesR lo hi = takeWhile (<= hi) (dropWhile (< lo) primes)
```

`takeWhile` 保证判断 $n$ 时只需求值到 $\sqrt n$ 附近，循环定义不会失控。

## 测试

```haskell
>>> take 8 primes
[2,3,5,7,11,13,17,19]
>>> primesR 10 20
[11,13,17,19]
>>> primesR 20 10
[]
```
