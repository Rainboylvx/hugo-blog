# P25 - 随机排列列表

> Generate a random permutation of the elements of a list
>
> 官方模块：`Problems.P25`　核心函数：`randomPermute`

---

[← P24 从 1..M 中随机选 N 个不同数](./p24) | [P26 生成所有 K 元组合 →](./p26)

---

## 题目描述

随机排列列表中的元素（Fisher-Yates 洗牌算法）。输入列表的所有元素必须出现在输出中，且每种排列等概率。

## 函数签名

```haskell
import System.Random (RandomGen)

randomPermute :: RandomGen g => [a] -> g -> ([a], g)
```

## 实现

### 方法一：基于 randomSelect

```haskell
import System.Random (RandomGen)

randomPermute :: RandomGen g => [a] -> g -> ([a], g)
randomPermute xs = randomSelect xs (length xs)
```

随机选 `length xs` 个元素，每次从剩余池中抽取不放回，结果恰好是原列表的一个随机排列。**利用 P23 一行搞定。**

### 方法二：Fisher-Yates 洗牌

```haskell
import System.Random (RandomGen, randomR)

randomPermute :: RandomGen g => [a] -> g -> ([a], g)
randomPermute xs gen = go (length xs) xs gen
  where
    go 0 acc gen = (acc, gen)
    go n acc gen =
      let (i, gen') = randomR (0, n - 1) gen
          swapped   = swap i (n - 1) acc
      in go (n - 1) swapped gen'

swap :: Int -> Int -> [a] -> [a]
swap i j xs
  | i == j    = xs
  | otherwise =
      [if k == i then xj else if k == j then xi else x
      | (k, x) <- zip [0..] xs]
  where
    xi = xs !! i
    xj = xs !! j
```

经典的 Fisher-Yates 算法，从后往前遍历，每步随机选一个未处理的元素与当前位置交换。

### 方法三：逐个插入

```haskell
import System.Random (RandomGen, randomR)

randomPermute :: RandomGen g => [a] -> g -> ([a], g)
randomPermute [] gen = ([], gen)
randomPermute (x:xs) gen =
  let (permuted, gen') = randomPermute xs gen
      (pos, gen'')     = randomR (0, length permuted) gen'
      (front, back)    = splitAt pos permuted
  in (front ++ x : back, gen'')
```

递归：先随机排列尾部，再随机将头元素插入结果中任意位置。思考过程更符合"分治"直觉，但效率不如 Fisher-Yates。

## Fisher-Yates 直观理解

```
初始: [a, b, c, d, e]

Step 1: 随机 i ∈ [0..4], 假设 i=2 → 交换位置 2 和 4 → [a, b, e, d, c]
Step 2: 随机 i ∈ [0..3], 假设 i=0 → 交换位置 0 和 3 → [d, b, e, a, c]
Step 3: 随机 i ∈ [0..2], 假设 i=1 → 交换位置 1 和 2 → [d, e, b, a, c]
Step 4: 随机 i ∈ [0..1], 假设 i=1 → 交换位置 1 和 1 → [d, e, b, a, c]
结束: [d, e, b, a, c]
```

## 测试

```haskell
>>> import Data.List (sort)
>>> import System.Random (mkStdGen)
>>> let (ys, _) = randomPermute [1..10] (mkStdGen 111)
>>> sort ys == [1..10]
True
>>> fst (randomPermute [] (mkStdGen 1))
[]
>>> sort (fst (randomPermute "aab" (mkStdGen 2)))
"aab"
```
