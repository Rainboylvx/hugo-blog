# P25 - 随机排列列表

> Generate a random permutation of a list
>
> 官方模块：`Problems.P25`　核心函数：`randomPermute`

## 题目描述

随机打乱列表，同时保留全部元素及其出现次数。对于含重复值的输入，不同位置仍被当作不同候选项。

## 函数签名

```haskell
import System.Random (RandomGen)

randomPermute :: RandomGen g => [a] -> g -> ([a], g)
```

## 思路与实现

P23 已经实现了“随机选择且不放回”。抽取数量等于列表长度时，结果显然就是一个随机排列。

```haskell
import System.Random (RandomGen)

randomPermute :: RandomGen g => [a] -> g -> ([a], g)
randomPermute xs = randomSelect xs (length xs)
```

## 测试

```haskell
import Data.List (sort)
import System.Random (mkStdGen)

>>> let (ys, _) = randomPermute [1..10] (mkStdGen 111)
>>> sort ys == [1..10]
True
>>> fst (randomPermute [] (mkStdGen 1))
[]
>>> sort (fst (randomPermute "aab" (mkStdGen 2)))
"aab"
```

## 要点总结

- 随机排列就是“不放回抽取全部元素”。
- 测试排列应比较排序后的多重集合，不应依赖具体随机顺序。
- 实现继承了 P23 的 $O(n^2)$ 列表删除成本。
