# P45 - 用二平方定理判定高斯素数

> Gaussian primes using the two-square theorem
>
> 官方模块：`Problems.P45`　核心函数：`isGaussianPrime'`

## 判定定理

高斯整数 $a+bi$ 为素数，当且仅当满足以下一种情况：

- $b=0$，且 $|a|$ 是模 4 余 3 的普通素数；
- $a=0$，且 $|b|$ 是模 4 余 3 的普通素数；
- $a,b$ 都非零，且 $a^2+b^2$ 是普通素数。

## 函数签名与实现

```haskell
import Data.Complex (Complex((:+)))

isGaussianPrime' :: Complex Integer -> Bool
isGaussianPrime' (a :+ 0) =
  abs a `mod` 4 == 3 && isPrime (abs a)
isGaussianPrime' (0 :+ b) =
  abs b `mod` 4 == 3 && isPrime (abs b)
isGaussianPrime' (a :+ b) = isPrime (a*a + b*b)
```

与 P44 的二维因子枚举相比，这里只做至多一次 $O(\sqrt n)$ 的普通素数判断。

## 测试

```haskell
>>> isGaussianPrime' (0 :+ 5)
False
>>> isGaussianPrime' (17 :+ 0)
False
>>> isGaussianPrime' (5 :+ 2)
True
>>> isGaussianPrime' (3 :+ 0)
True
```

## 要点总结

P44 从定义出发，P45 从结构定理出发。二者结果相同，但后者展示了数学结论如何直接改变算法复杂度。
