# P57 - 二叉搜索树

> Binary search trees

> 官方模块：`Problems.P57`　核心函数：`construct`, `addedTo`

## 题目描述

按输入顺序把元素插入二叉搜索树。小于当前节点的值进入左子树，其余值进入右子树，因此重复值也会保留。

## 函数签名

```haskell
addedTo :: Ord a => a -> Tree a -> Tree a
construct :: Ord a => [a] -> Tree a
```

## 实现

### 方法一：递归插入 + foldl

```haskell
addedTo :: Ord a => a -> Tree a -> Tree a
addedTo x Empty = leaf x
addedTo x (Branch y left right)
  | x < y     = Branch y (addedTo x left) right
  | otherwise = Branch y left (addedTo x right)

construct :: Ord a => [a] -> Tree a
construct = foldl (flip addedTo) Empty
```

标准 BST 插入：比较、递归、重建。`construct` 用 `foldl` 逐个插入。

### 方法二：递归插入（if 风格）

```haskell
addedTo :: Ord a => a -> Tree a -> Tree a
addedTo x Empty = leaf x
addedTo x t@(Branch y l r) =
  if x < y then Branch y (addedTo x l) r
           else Branch y l (addedTo x r)
```

用 `if` 代替 guard，同一行内完成分支。

### 方法三：construct 用尾递归 + 累加器

```haskell
construct :: Ord a => [a] -> Tree a
construct = go Empty
  where
    go acc []     = acc
    go acc (x:xs) = go (addedTo x acc) xs
```

显式 `go` 函数控制递归，与 `foldl` 等价但更显式。

## 测试

```haskell
>>> construct "BSTree"
Branch 'B' (Branch 'B' (leaf 'B') Empty) (Branch 'T' (Branch 'S' Empty (Branch 'T' (leaf 'e') (leaf 'r'))) Empty)
>>> symmetric (construct "abccba")
True
```
