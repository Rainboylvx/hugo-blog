# P53 - 归结原理

> Resolution rule for theorem proving

## 题目描述

用归结原理判断一组公理能否推出一个结论。即将 $A \land \neg C$ 转为 CNF，反复应用归结规则直到推出空子句（矛盾）或无法继续。

## 数据类型与原理

```haskell
import qualified Data.Set as Set

type Clause = Set.Set Formula
```

要证明公理集合 $A$ 蕴含结论 $C$，把 $A\land\neg C$ 转成 CNF，然后尝试推出空子句。若推出矛盾，原命题成立。

## 函数签名

```haskell
isTheorem :: [Formula] -> Formula -> Bool
```

## 实现

### 方法一：标准归结算法

```haskell
isTheorem :: [Formula] -> Formula -> Bool
isTheorem axioms conjecture = refute initial
  where
    Conjoin clauses = toConjunctiveNormalForm
      (Conjoin (Complement conjecture : axioms))
    initial = Set.fromList [Set.fromList (disjuncts c) | c <- clauses]

    disjuncts (Disjoin fs) = fs
    disjuncts f            = [f]

    refute clauses
      | Set.member Set.empty clauses = True
      | null newPairs                 = False
      | otherwise                     = refute (Set.union clauses new)
      where
        pairs = [(c1, c2) | c1 <- Set.toList clauses, c2 <- Set.toList clauses, c1 < c2]
        new = Set.fromList [resolve c1 c2 | (c1, c2) <- pairs, resolvable c1 c2]
        newPairs = Set.toList new

    resolve c1 c2 = Set.union (Set.delete p c1) (Set.delete (Complement p) c2)
      where p = head [l | l <- Set.toList c1, Complement l `Set.member` c2]

    resolvable c1 c2 = not . null $ [l | l <- Set.toList c1, Complement l `Set.member` c2]
```

从 CNF 子句集开始，反复寻找互补文字对并应用归结规则。当空子句出现时，结论得证。

### 方法二：真值表验证（穷举法）

```haskell
isTheorem :: [Formula] -> Formula -> Bool
isTheorem axioms conjecture =
  all valid (environments (variables (Conjoin (conjecture : axioms))))
  where
    valid env = evaluate env (Conjoin axioms) `implies` evaluate env conjecture
    implies a b = not a || b
```

对所有可能的环境（真值赋值），检查公理成立时结论是否也成立。本质上做了一次穷举验证——变量数少时可用，变量多时组合爆炸。

## 方法对比

| 方法 | 特点 | 适用场景 |
|------|------|----------|
| 归结原理 | 符号推导，可处理大变量集 | 标准做法 |
| 真值表穷举 | 枚举所有环境 | 变量很少（<10）时验证用 |

## 测试

```haskell
>>> let p = Variable "p"; q = Variable "q"
>>> isTheorem [Disjoin [Complement p, q], Disjoin [Complement q, p]] (Disjoin [Complement p, Complement (Complement p)])
False
```
