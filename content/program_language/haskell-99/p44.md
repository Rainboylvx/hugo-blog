# P44 - 朴素判定高斯素数

> Determine whether a Gaussian integer is prime
>
> 官方模块：`Problems.P44`　核心函数：`isGaussianPrime`

---

[← P43 高斯整数整除](./p43) | [P45 二平方定理判定 →](./p45)

---

## 题目描述

高斯整数的单位元为 $\pm1,\pm i$。若一个非零非单位高斯整数除单位元和自身的相伴元外没有其他因子，它就是高斯素数。

## 函数签名

```haskell
isGaussianPrime :: Complex Integer -> Bool
```

## 实现

### 方法一：枚举候选因子

```haskell
import Data.Complex (Complex((:+)))

isGaussianPrime :: Complex Integer -> Bool
isGaussianPrime z
  | nz <= 1   = False
  | otherwise = not (any (z `gaussianDividesBy`) candidates)
  where
    nz = normSquared z
    limit = integerSqrt (nz - 1)
    candidates =
      [x :+ y | x <- [-limit..limit], y <- [-limit..limit]
              , let nd = x*x + y*y, nd > 1, nd < nz]

normSquared :: Complex Integer -> Integer
normSquared (a :+ b) = a*a + b*b

integerSqrt :: Integer -> Integer
integerSqrt n = last (takeWhile (\x -> x*x <= n) [0..])
```

用范数 $N(a+bi)=a^2+b^2$ 衡量大小。真正的非单位因子 `d` 必须满足 $1<N(d)<N(z)$，枚举这个有限区域并复用 P43 的整除判断。

### 方法二：只枚举上半平面

```haskell
isGaussianPrime :: Complex Integer -> Bool
isGaussianPrime z
  | nz <= 1   = False
  | otherwise = not (any (z `gaussianDividesBy`) candidates)
  where
    nz = normSquared z
    limit = integerSqrt (nz - 1)
    candidates =
      [(x :+ y) | x <- [0..limit], y <- [-limit..limit]
                , let nd = x*x + y*y, nd > 1, nd < nz
                , nd <= nz - 1]
```

利用对称性只枚举 $x \ge 0$ 的一半区间，减少约一半的检查量。

### 方法三：预筛法优化

```haskell
isGaussianPrime :: Complex Integer -> Bool
isGaussianPrime z
  | nz <= 1   = False
  | otherwise = null [d | d <- candidateNorms
                        , let ds = integerSqrt d
                        , x <- [-ds..ds], y <- [-ds..ds]
                        , x*x + y*y == d
                        , z `gaussianDividesBy` (x :+ y)]
  where
    nz = normSquared z
    candidateNorms = [d | d <- [2..nz-1], nz `mod` d == 0]
```

先枚举范数的因数，再检查是否有对应的高斯整数因子。减少了对 `gaussianDividesBy` 的无效调用。

## 方法对比

| 方法 | 枚举量 | 特点 |
|------|--------|------|
| 全枚举 | $(2L+1)^2$ | 最直接，易懂 |
| 上半平面 | $(L+1)(2L+1)$ | 利用对称性优化 |
| 范数因数预筛 | 取决于因数个数 | 对合数最优 |

## 测试

```haskell
>>> isGaussianPrime (0 :+ 5)
False
>>> isGaussianPrime (17 :+ 0)
False
>>> isGaussianPrime (5 :+ 2)
True
```
