# P44 - 朴素判定高斯素数

> Determine whether a Gaussian integer is prime
>
> 官方模块：`Problems.P44`　核心函数：`isGaussianPrime`

---

[← P43 高斯整数整除](./p43) | [P45 二平方定理判定 →](./p45)

## 题目描述

高斯整数的单位元为 $\pm1,\pm i$。若一个非零非单位高斯整数除单位元和自身的相伴元外没有其他因子，它就是高斯素数。

## 思路与实现

用范数 $N(a+bi)=a^2+b^2$ 衡量大小。真正的非单位因子 `d` 必须满足 $1<N(d)<N(z)$，枚举这个有限区域并复用 P43 的整除判断。

```haskell
import Data.Complex (Complex((:+)))

isGaussianPrime :: Complex Integer -> Bool
isGaussianPrime z
  | nz <= 1   = False
  | otherwise = not (any (z `gaussianDividesBy`) candidates)
  where
    nz = normSquared z
    limit = integerSqrt (nz - 1)
    candidates =
      [x :+ y | x <- [-limit..limit], y <- [-limit..limit]
              , let nd = x*x + y*y, nd > 1, nd < nz]

normSquared :: Complex Integer -> Integer
normSquared (a :+ b) = a*a + b*b

integerSqrt :: Integer -> Integer
integerSqrt n = last (takeWhile (\x -> x*x <= n) [0..])
```

这个实现强调定义，枚举量很大。P45 会利用数论定理把判定降为普通素数测试。

## 测试

```haskell
>>> isGaussianPrime (0 :+ 5)
False
>>> isGaussianPrime (17 :+ 0)
False
>>> isGaussianPrime (5 :+ 2)
True
```
