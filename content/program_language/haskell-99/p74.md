# P74 - 不用 do 记法编写 IO

> IO monad without do notation
>
> 官方模块：`Problems.P74`　核心函数：`askGoldbach`

---

[← P73 S 表达式](./p73) | [P75 Maybe Monad →](./p75)

## 题目描述

从输入句柄读取一个偶数，调用 P40 分解，再把 `n=a+b` 写到输出句柄。要求不用 `do`，直接使用 `(>>=)` 和 `(>>)`。

## 实现

```haskell
import System.IO (Handle, hGetLine, hPutStr, hPutStrLn)

askGoldbach :: Handle -> Handle -> IO ()
askGoldbach input output =
  hGetLine input >>= \text ->
    let n = read text :: Integer
        (a,b) = goldbach n
    in hPutStr output (show n ++ "=")
       >> hPutStr output (show a ++ "+")
       >> hPutStrLn output (show b)
```

`(>>=)` 把读取到的字符串交给后续函数；`(>>)` 只关心动作顺序，忽略前一个动作的 `()` 结果。

## 与 do 记法对照

```haskell
askGoldbach' input output = do
  text <- hGetLine input
  let n = read text :: Integer
      (a,b) = goldbach n
  hPutStrLn output (show n ++ "=" ++ show a ++ "+" ++ show b)
```

## 测试

使用内容为 `104` 的输入句柄时，输出为：

```text
104=3+101
```

`return` 在这里不是命令式语言的“返回语句”，而是把普通值放入某个 Monad 的函数。
