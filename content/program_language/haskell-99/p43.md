# P43 - 高斯整数整除

> Determine divisibility between Gaussian integers
>
> 官方模块：`Problems.P43`　核心函数：`gaussianDividesBy`

---

## 核心公式

高斯整数形如 $a+bi$。判断 $x/y$ 的实部与虚部是否都是整数，可将分母实数化：

$$
\frac{a+bi}{c+di}
=\frac{(ac+bd)+(bc-ad)i}{c^2+d^2}.
$$

## 函数签名与实现

```haskell
import Data.Complex (Complex((:+)))

gaussianDividesBy :: Complex Integer -> Complex Integer -> Bool
gaussianDividesBy _ (0 :+ 0) = False
gaussianDividesBy (a :+ b) (c :+ d) =
  realNumerator `mod` denominator == 0
  && imagNumerator `mod` denominator == 0
  where
    realNumerator = a * c + b * d
    imagNumerator = b * c - a * d
    denominator   = c * c + d * d
```

`Complex Integer` 没有普通复数除法；直接检查两个分子能否被范数整除，恰好避免了浮点误差。

## 测试

```haskell
>>> (10 :+ 0) `gaussianDividesBy` (2 :+ 0)
True
>>> (10 :+ 0) `gaussianDividesBy` (0 :+ 2)
True
>>> (5 :+ 2) `gaussianDividesBy` (2 :+ (-1))
False
```
