# P43 - 高斯整数整除

> Determine whether a Gaussian integer divides another
>
> 官方模块：`Problems.P43`　核心函数：`gaussianDividesBy`

---

[← P42 模乘逆元](./p42) | [P44 朴素判定高斯素数 →](./p44)

---

## 题目描述

判断一个高斯整数是否整除另一个高斯整数。高斯整数形如 $a+bi$，判别 $x/y$ 的实部与虚部是否都是整数。

## 函数签名

```haskell
gaussianDividesBy :: Complex Integer -> Complex Integer -> Bool
```

`Complex Integer` 没有普通复数除法；直接检查两个分子能否被范数整除，恰好避免了浮点误差。

## 实现

### 方法一：分母实数化

```haskell
import Data.Complex (Complex((:+)))

gaussianDividesBy :: Complex Integer -> Complex Integer -> Bool
gaussianDividesBy _ (0 :+ 0) = False
gaussianDividesBy (a :+ b) (c :+ d) =
  realNumerator `mod` denominator == 0
  && imagNumerator `mod` denominator == 0
  where
    realNumerator = a * c + b * d
    imagNumerator = b * c - a * d
    denominator   = c * c + d * d
```

将分母实数化 $\frac{a+bi}{c+di} = \frac{(ac+bd)+(bc-ad)i}{c^2+d^2}$，检查两个分子能否被范数整除。

### 方法二：枚举高斯整数商

```haskell
import Data.Complex (Complex((:+)))

gaussianDividesBySearch :: Complex Integer -> Complex Integer -> Bool
gaussianDividesBySearch _ (0 :+ 0) = False
gaussianDividesBySearch target divisor =
  any ((== target) . multiply divisor) candidates
  where
    targetNorm = norm target
    candidates = [r :+ i | r <- [-targetNorm..targetNorm]
                           , i <- [-targetNorm..targetNorm]]

    norm (a :+ b) = a * a + b * b

    multiply (a :+ b) (c :+ d) =
      (a * c - b * d) :+ (a * d + b * c)
```

若 `divisor` 整除 `target`，就存在高斯整数 $q$ 使 `divisor * q == target`。商的范数不会超过被除数的范数，所以在这个有限范围内枚举实部和虚部即可。这个版本很慢，但它直接按照“存在整数商”的定义求解，适合验证方法一。

## 方法对比

| 方法 | 特点 |
|------|------|
| 分母实数化 | O(1) 代数判定，推荐 |
| 枚举高斯整数商 | 定义直译，适合小数值交叉验证 |

## 测试

```haskell
>>> (10 :+ 0) `gaussianDividesBy` (2 :+ 0)
True
>>> (10 :+ 0) `gaussianDividesBy` (0 :+ 2)
True
>>> (5 :+ 2) `gaussianDividesBy` (2 :+ (-1))
False
>>> gaussianDividesBySearch (10 :+ 0) (0 :+ 2)
True
>>> gaussianDividesBy (0 :+ 0) (0 :+ 0)
False
```
