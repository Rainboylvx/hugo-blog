# P43 - 高斯整数整除

> Determine whether a Gaussian integer divides another
>
> 官方模块：`Problems.P43`　核心函数：`gaussianDividesBy`

---

[← P42 模乘逆元](./p42) | [P44 朴素判定高斯素数 →](./p44)

---

## 题目描述

判断一个高斯整数是否整除另一个高斯整数。高斯整数形如 $a+bi$，判别 $x/y$ 的实部与虚部是否都是整数。

## 函数签名

```haskell
gaussianDividesBy :: Complex Integer -> Complex Integer -> Bool
```

`Complex Integer` 没有普通复数除法；直接检查两个分子能否被范数整除，恰好避免了浮点误差。

## 实现

### 方法一：分母实数化

```haskell
import Data.Complex (Complex((:+)))

gaussianDividesBy :: Complex Integer -> Complex Integer -> Bool
gaussianDividesBy _ (0 :+ 0) = False
gaussianDividesBy (a :+ b) (c :+ d) =
  realNumerator `mod` denominator == 0
  && imagNumerator `mod` denominator == 0
  where
    realNumerator = a * c + b * d
    imagNumerator = b * c - a * d
    denominator   = c * c + d * d

将分母实数化 $\frac{a+bi}{c+di} = \frac{(ac+bd)+(bc-ad)i}{c^2+d^2}$，检查两个分子能否被范数整除。

### 方法二：辅函数分步骤

```haskell
import Data.Complex (Complex((:+)))

gaussianDividesBy :: Complex Integer -> Complex Integer -> Bool
gaussianDividesBy _ (0 :+ 0) = False
gaussianDividesBy x y = divides num $ denom y
  where
    num = x * conjugate y
    denom (c :+ d) = c * c + d * d

conjugate (c :+ d) = c :+ (-d)

divides (a :+ b) d = a `mod` d == 0 && b `mod` d == 0
```

利用共轭：$x \bar y = (a+bi)(c-di) = (ac+bd)+(bc-ad)i$，分母为 $c^2+d^2$。

### 方法三：inline 条件版本

```haskell
gaussianDividesBy :: Complex Integer -> Complex Integer -> Bool
gaussianDividesBy (a :+ b) (c :+ d) = (a*c + b*d) `mod` n == 0 && (b*c - a*d) `mod` n == 0
  where n = c*c + d*d
```

一行版，去掉了注释和 `where`。代码最短但可读性降低。

## 方法对比

| 方法 | 特点 |
|------|------|
| 分母实数化 | 直接计算，推荐 |
| 共轭辅助函数 | 语义更清晰 |
| inline 一行版 | 最紧凑 |

## 测试

```haskell
>>> (10 :+ 0) `gaussianDividesBy` (2 :+ 0)
True
>>> (10 :+ 0) `gaussianDividesBy` (0 :+ 2)
True
>>> (5 :+ 2) `gaussianDividesBy` (2 :+ (-1))
False
```
