[Ninety-Nine Haskell Problems](https://ninetynine.haskell.chungyc.org/) 是 Haskell 社区经典的 99 道练习题，源自 Prolog 99 题的 Haskell 移植版。每个题目都有一个函数骨架，你只需要实现它，然后跑预置测试验证。

这套题的独特价值在于：**难度梯度设计得好**，从最基础的列表递归一路做到图同构、Huffman 编码、数独求解。把 LYAH 学到的语法点串成实战能力。

## 第一阶段：列表基础（P01-P10）

**覆盖 LYAH：Ch1-5**

| 题号 | 题目 | 核心语法 | 推荐 |
|------|------|----------|------|
| [P01](./p01) | 取列表最后一个元素 | 递归基条件、`Maybe` | ★ |
| [P02](./p02) | 取列表倒数第二个元素 | 多条件递归 | ★ |
| [P03](./p03) | 取列表第 k 个元素 | 下标计数 | ★ |
| [P04](./p04) | 求列表长度 | 尾递归、fold | ★ |
| [P05](./p05) | 反转列表 | 递归 + `++` vs 尾递归优化 | ★ |
| [P06](./p06) | 判断回文 | `==`、列表比较 | ★ |
| [P07](./p07) | 展平嵌套列表 | 自定义 ADT、递归遍历 | ★ |
| [P08](./p08) | 去除连续重复元素 | `Eq` 约束、guard | ★ |
| [P09](./p09) | 将连续重复元素打包成子列表 | `group` 的手动实现 | ★★ |
| [P10](./p10) | 游程编码 (run-length encoding) | 结合 P09、tuple | ★★ |

> [!TIP] 阶段要点
> P01-P06 是热身，闭着眼也要能写出来。P07 让你第一次接触自定义 `data` 类型（`NestedList`），为后面树和图打基础。P08-P10 是 `group` 的三种变体，理解它们对第 7 章 `Data.List` 模块的理解有质的提升。

| [P15](./p15) | 每个元素重复 N 次 | `concatMap` | ★ |
| [P16](./p16) | 每隔 N 个删除一个元素 | `zip` + 取模计数 | ★★ |

| [P18](./p18) | 提取子列表 [i, k] | `take`/`drop` 组合 | ★★ |

## 第三阶段：组合与随机（P21-P28）

**覆盖 LYAH：Ch5-7**

| 题号 | 题目 | 核心语法 | 推荐 |
|------|------|----------|------|

| [P22](./p22) | 生成整数范围 | `enumFromTo` 或递归 | ★ |
| [P23](./p23) | 从列表随机选 N 个元素 | `RandomGen`、ST monad | ★★★ |
| [P24](./p24) | 从 1..M 中随机选 N 个不同数 | P23 的应用 | ★★ |

| [P28](./p28) | 按子列表长度排序 / 按出现频率排序 | `sortOn`、`group . sort`、`comparing` | ★★ |

> [!TIP] 阶段要点
> P26 是经典回溯题，用 list comprehension 写比 C++ 的 for 循环嵌套清爽太多。P23-P25 涉及随机性，Haskell 的纯函数随机数（State monad）跟命令式语言完全不同，这一关是**思维转换节点**。如果卡住，可以暂时跳过 P23-P25 先往后走，回来再看。

## 实操建议

### 环境搭建

```bash
cabal install ninetynine
```

安装后得到 `Problems.*` 模块，每个模块里有 `myLast`、`myButLast` 这样的函数骨架。

### 学习节奏

- **刚学完 LYAH Ch1-5**：做 P01-P10，边做边巩固递归和模式匹配
- **学到 LYAH Ch7**：做 P11-P28，练习 `take`/`drop`/`group` 组合
- **学到 LYAH Ch8**：做 P54-P69，树和自定义类型
- **学到 LYAH Ch12-13**：做 P46-P53 + P74-P79，逻辑、编码和 Monad
- **想要挑战**：P90 八皇后（掌握 list monad 后有惊喜）、P40 无限素数流（体验惰性求值）

### 哪些可以跳过

- P23-P25（随机性）：涉及 `RandomGen` 和 State monad，初学者杀手
- P43-P45（高斯整数扩展）：除非对数学感兴趣
- P85（图同构）：NP-hard 问题，纯粹学术挑战
- P98-P99（大型工程）：需要大量调试精力
