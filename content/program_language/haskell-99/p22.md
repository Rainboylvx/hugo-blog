# P22 - 生成整数范围

> Create a list containing all integers within a given range
>
> 官方模块：`Problems.P22`　核心函数：`range`

## 题目描述

给定整数下界和上界，生成包含两个端点的递增整数列表。

## 函数签名

```haskell
range :: Int -> Int -> [Int]
```

## 思路与实现

Haskell 的枚举语法已经准确表达了这个需求：

```haskell
range :: Int -> Int -> [Int]
range from to = [from .. to]
```

也可以用 `iterate` 展示惰性列表的构造过程。`iterate (+ 1) from` 先生成无限递增序列，再截断到上界。

```haskell
range' :: Int -> Int -> [Int]
range' from to = takeWhile (<= to) (iterate (+ 1) from)
```

## 测试

```haskell
>>> range 4 9
[4,5,6,7,8,9]
>>> range (-2) 2
[-2,-1,0,1,2]
>>> range 3 2
[]
```

## 要点总结

- `[from .. to]` 包含两个端点。
- 当下界大于上界时，默认步长仍为 1，因此结果为空。
- `iterate` 先构造无限列表，惰性求值使 `takeWhile` 可以安全消费它。
