# P84 - 用 Prim 算法构造最小生成树

> Construct a minimum spanning tree
>
> 官方模块：`Problems.P84`　核心函数：`minimumSpanningTree`

## 算法

从任意顶点开始，反复选择连接“树内”和“树外”的最小权边。加入这条边不会产生环；连通图在加入 $n-1$ 条边后覆盖全部顶点。

## 实现

```haskell
minimumSpanningTree :: G -> Map.Map Edge Int -> G
minimumSpanningTree graph weights
  | Set.null vs = graph
  | otherwise   = grow (Set.singleton (Set.findMin vs)) []
  where
    (vs,es) = canonical graph
    grow inside chosen
      | inside == vs = toG (Lists (Set.toList vs, chosen))
      | null crossing = toG (Lists ([],[]))
      | otherwise =
          let edge@(u,v) = minimumByWeight crossing
              newVertex = if Set.member u inside then v else u
          in grow (Set.insert newVertex inside) (edge : chosen)
      where
        crossing = [e | e@(u,v) <- Set.toList es
                       , Set.member u inside /= Set.member v inside]

    minimumByWeight = minimumBy
      (comparing (\edge -> Map.findWithDefault 0 edge weights))
```

需要导入 `Data.List (minimumBy)` 和 `Data.Ord (comparing)`。权重映射的键必须使用 P80 的规范化无向边。

## 测试

```haskell
>>> let g = toG (Paths [[1,2,3],[1,3]])
>>> let w = Map.fromList [((1,2),5),((2,3),1),((1,3),2)]
>>> canonical (minimumSpanningTree g w)
(fromList [1,2,3],fromList [(1,3),(2,3)])
```

当前版本每轮线性扫描所有边，复杂度 $O(nm)$；使用优先队列可优化到 $O(m\log n)$。
