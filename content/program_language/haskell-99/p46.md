# P46 - 二元逻辑真值表

> Truth tables for logical expressions of two variables
>
> 官方模块：`Problems.P46`　核心函数：`table`

---

[← P45 二平方定理](./p45) | [P47 通用逻辑门 →](./p47)

---

## 题目描述

不用内置布尔运算符定义常见二元逻辑函数，并为任意二元布尔表达式生成四行真值表。

## 函数签名

```haskell
type BoolFunc = Bool -> Bool -> Bool

and', or', nand', nor', xor', impl', equ' :: BoolFunc
table :: BoolFunc -> [(Bool, Bool, Bool)]
```

## 实现

### 方法一：模式匹配 + 列表推导

```haskell
and' True True = True
and' _    _    = False

or' False False = False
or' _     _     = True

nand' True True = False
nand' _    _    = True

nor' False False = True
nor' _     _     = False

xor' True False = True
xor' False True = True
xor' _     _    = False

impl' True False = False
impl' _    _     = True

equ' True  True  = True
equ' False False = True
equ' _     _     = False

table :: BoolFunc -> [(Bool, Bool, Bool)]
table f = [(a, b, f a b) | a <- bools, b <- bools]
  where bools = [False, True]
```

### 方法二：查真值表

```haskell
and'  = fromTruthTable [False, False, False, True]
or'   = fromTruthTable [False, True, True, True]
nand' = fromTruthTable [True, True, True, False]
nor'  = fromTruthTable [True, False, False, False]
xor'  = fromTruthTable [False, True, True, False]
impl' = fromTruthTable [True, True, False, True]
equ'  = fromTruthTable [True, False, False, True]

fromTruthTable :: [Bool] -> BoolFunc
fromTruthTable values a b = values !! (2 * bit a + bit b)
  where
    bit False = 0
    bit True  = 1
```

四个表项依次对应 `(False,False)`、`(False,True)`、`(True,False)`、`(True,True)`。这个版本没有使用待实现的内置二元布尔运算符，而是把逻辑函数当作有限查找表。

### 方法三：显式列出输入顺序

```haskell
table :: BoolFunc -> [(Bool, Bool, Bool)]
table f = map evaluate
  [(False, False), (False, True), (True, False), (True, True)]
  where
    evaluate (a, b) = (a, b, f a b)
```

显式输入表可以确保行顺序与查表版本一致，也避免嵌套列表推导。

## 测试

```haskell
>>> table (\a b -> and' a (or' a b))
[(False,False,False),(False,True,False),(True,False,True),(True,True,True)]
>>> xor' True False
True
>>> impl' True False
False
```
