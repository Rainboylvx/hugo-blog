# P46 - 二元逻辑真值表

> Truth tables for logical expressions of two variables
>
> 官方模块：`Problems.P46`　核心函数：`table`

---

## 题目描述

不用内置布尔运算符定义常见二元逻辑函数，并为任意二元布尔表达式生成四行真值表。

## 函数签名

```haskell
type BoolFunc = Bool -> Bool -> Bool

and', or', nand', nor', xor', impl', equ' :: BoolFunc
table :: BoolFunc -> [(Bool, Bool, Bool)]
```

## 实现

### 方法一：模式匹配 + 列表推导

```haskell
and' True True = True
and' _    _    = False

or' False False = False
or' _     _     = True

nand' True True = False
nand' _    _    = True

nor' False False = True
nor' _     _     = False

xor' True False = True
xor' False True = True
xor' _     _    = False

impl' True False = False
impl' _    _     = True

equ' True  True  = True
equ' False False = True
equ' _     _     = False

table :: BoolFunc -> [(Bool, Bool, Bool)]
table f = [(a, b, f a b) | a <- bools, b <- bools]
  where bools = [False, True]
```

### 方法二：用内置运算符直接定义

```haskell
and'  = (&&)
or'   = (||)
nand' a b = not (a && b)
nor'  a b = not (a || b)
xor'  a b = a /= b
impl' a b = not a || b
equ'  a b = a == b
```

利用 Haskell 内置的布尔运算符，一行一个定义。与模式匹配版等价但更简洁。

### 方法三：table 的递归版

```haskell
table :: BoolFunc -> [(Bool, Bool, Bool)]
table f = [(a, b, f a b) | a <- [False, True], b <- [False, True]]
```

明确了两个变量的取值范围。

## 测试

```haskell
>>> table (\a b -> and' a (or' a b))
[(False,False,False),(False,True,False),(True,False,True),(True,True,True)]
>>> xor' True False
True
>>> impl' True False
False
```
