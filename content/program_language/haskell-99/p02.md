# P02 - 取列表倒数第二个元素

> Penultimate element of a list
>
> 官方模块: `Problems.P02`
> 核心函数: `myButLast`

## 题目描述

找出列表的倒数第二个元素（penultimate element）。如果列表不足两个元素，返回 `Nothing`。

## 函数签名

```haskell
myButLast :: [a] -> Maybe a
```

## 思路

和 P01 类似，但需要多一个元素的前瞻。模式匹配时有三种情况：

- 空列表 `[]` 或单元素 `[_]`——不足两个，返回 `Nothing`
- 正好两个元素 `[x, _]`——那 `x` 就是倒数第二个
- 至少三个元素 `(_:xs)`——递归处理 `xs`，把问题规模缩小

## 实现

### 方法一：标准递归

```haskell
myButLast :: [a] -> Maybe a
myButLast []       = Nothing
myButLast [_]      = Nothing
myButLast [x, _]   = Just x
myButLast (_:xs)   = myButLast xs
```

前两个模式处理不足两元素的 case；`[x, _]` 匹配正好两个元素，返回第一个；`(_:xs)` 丢弃头部，在更短的列表上继续找。

### 方法二：利用 `reverse` 和 `listToMaybe`

```haskell
import Data.Maybe (listToMaybe)

myButLast :: [a] -> Maybe a
myButLast [] = Nothing
myButLast xs = listToMaybe . drop 1 . reverse $ xs
```

`reverse` 反转后倒数第二个变成正数第二个，`drop 1` 扔掉第一个，再取 `listToMaybe`。

### 方法三：使用标准库 `init` + `last`

```haskell
myButLast :: [a] -> Maybe a
myButLast [] = Nothing
myButLast [_] = Nothing
myButLast xs = Just . last . init $ xs
```

`init` 去掉最后一个元素，`last` 取结果列表的最后一个（即原列表的倒数第二个）。

## 测试

```haskell
>>> myButLast [1,2,3,4]
Just 3

>>> myButLast ['a'..'z']
Just 'y'

>>> myButLast ['a']
Nothing

>>> myButLast []
Nothing
```
