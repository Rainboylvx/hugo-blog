# P60 - 给定节点数的高度平衡二叉树

> Construct height-balanced binary trees with a given number of nodes
>
> 官方模块：`Problems.P60`　核心函数：`heightBalancedTreesWithNodes`

---

[← P59 高度平衡二叉树](./p59) | [P61 收集叶子节点 →](./p61)

---

## 题目描述

给定节点数 $n$，找出所有恰好有 $n$ 个节点的高度平衡二叉树。

## 函数签名

```haskell
heightBalancedTreesWithNodes :: Int -> [Tree ()]
```

## 实现

### 方法一：枚举高度 + 过滤

```haskell
heightBalancedTreesWithNodes :: Int -> [Tree ()]
heightBalancedTreesWithNodes n
  | n < 0     = []
  | otherwise =
      [tree | h <- [0..n]
            , tree <- heightBalancedTrees h
            , treeSize tree == n]
```

先算出所有可能的高度 $h$（不超过 $n$），用 P59 生成该高度的所有树，再过滤出节点数正好为 $n$ 的。

### 方法二：用 minNodes 缩小范围

```haskell
heightBalancedTreesWithNodes :: Int -> [Tree ()]
heightBalancedTreesWithNodes n
  | n < 0     = []
  | otherwise =
      [tree | h <- [minH..maxH]
            , tree <- heightBalancedTrees h
            , treeSize tree == n]
  where
    minH = ceiling (logBase 2 (fromIntegral (n + 1))) -- 完全二叉树对应最矮高度
    maxH = head [h | h <- [0..], minNodes h > n] - 1   -- 高度上界
```

用 `minNodes`（P59 的辅助函数）提前确定可能的高度范围，减少不必要的枚举。

## 测试

```haskell
>>> length (heightBalancedTreesWithNodes 4)
4
>>> all ((== 4) . treeSize) (heightBalancedTreesWithNodes 4)
True
>>> heightBalancedTreesWithNodes 0
[Empty]
```
