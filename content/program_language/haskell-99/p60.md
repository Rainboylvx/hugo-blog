# P60 - 指定节点数的高度平衡二叉树

> Height-balanced trees with a given number of nodes
>
> 官方模块：`Problems.P60`　核心函数：`heightBalancedTreesWithNodes`

## 思路

节点数为 $n$ 的树高度不可能超过 $n$。因此可以生成高度 $0$ 到 $n$ 的所有高度平衡树，再按节点数过滤。这个版本偏重正确性；更高效的做法是用最少节点递推缩小高度范围。

## 实现

```haskell
heightBalancedTreesWithNodes :: Int -> [Tree ()]
heightBalancedTreesWithNodes n
  | n < 0     = []
  | otherwise =
      [tree | h <- [0..n]
            , tree <- heightBalancedTrees h
            , treeSize tree == n]
```

## 高度边界优化

高度 $h$ 的高度平衡树最少节点数满足：

```haskell
minNodes 0 = 0
minNodes 1 = 1
minNodes h = 1 + minNodes (h - 1) + minNodes (h - 2)
```

实际实现可只枚举满足 `minNodes h <= n <= 2^h - 1` 的高度。

## 测试

```haskell
>>> length (heightBalancedTreesWithNodes 4)
4
>>> all ((== 4) . treeSize) (heightBalancedTreesWithNodes 4)
True
>>> heightBalancedTreesWithNodes 0
[Empty]
```
