# P17 - 在 N 处切分列表

> Split a list into two parts
>
> 官方模块: `Problems.P17`
> 核心函数: `split`

---

## 题目描述

在指定的位置 N 处将列表切分为两个子列表。前一部分包含前 N 个元素，后一部分包含剩余元素。

## 函数签名

```haskell
split :: [a] -> Int -> ([a], [a])
```

## 思路

- `take n xs` 取前 N 个
- `drop n xs` 取剩余部分

## 实现

### 方法一：`take` + `drop`

```haskell
split :: [a] -> Int -> ([a], [a])
split xs n = (take n xs, drop n xs)
```

标准库 `splitAt` 做的就是这件事，这就是它的实现方式。

如果 N 大于列表长度，`take` 返回整个列表，`drop` 返回空列表，行为安全。

### 方法二：递归

```haskell
split :: [a] -> Int -> ([a], [a])
split xs 0 = ([], xs)
split [] _ = ([], [])
split (x:xs) n = let (first, second) = split xs (n - 1)
                 in (x : first, second)
```

不用 `take`/`drop`，纯递归实现。N 递减到 0 时停止取元素。

### 方法三：使用标准库

```haskell
import Data.List (splitAt)

split :: [a] -> Int -> ([a], [a])
split = splitAt
```

标准库的 `splitAt` 就是 P17 的官方解法。

## 与 `splitAt` 的关系

实际上 P17 的 `split` 就是标准库 `splitAt` 的重新实现。`splitAt` 在 `GHC.List` 中的定义就是：

```haskell
splitAt n xs = (take n xs, drop n xs)
```

## 测试

```haskell
>>> split "abcdefghik" 3
("abc","defghik")

>>> split [1,2,3] 5
([1,2,3],[])

>>> split [] 3
([],[])
```
