# P67 - 二叉树的括号字符串表示

> A string representation of binary trees

## 编码规则

叶子只写字符；非叶节点写成 `根(左,右)`；空子树写为空串。例如 `a(b(d,e),c(,f(g,)))`。

## 实现

### 方法一：手写递归下降

```haskell
treeToString :: Tree Char -> String
treeToString Empty = ""
treeToString (Branch c Empty Empty) = [c]
treeToString (Branch c left right) =
  c : '(' : treeToString left ++ "," ++ treeToString right ++ ")"

stringToTree :: String -> Maybe (Tree Char)
stringToTree "" = Just Empty
stringToTree input = do
  (tree, rest) <- parseTree input
  if null rest then Just tree else Nothing

parseTree :: String -> Maybe (Tree Char, String)
parseTree [] = Nothing
parseTree (c:'(':rest) = do
  (left, afterLeft) <- parseOptional rest
  (right, afterRight) <- case afterLeft of
    ',':xs -> parseOptional xs
    _      -> Nothing
  case afterRight of
    ')':xs -> Just (Branch c left right, xs)
    _      -> Nothing
parseTree (c:rest) = Just (leaf c, rest)

parseOptional :: String -> Maybe (Tree Char, String)
parseOptional input@(',':_) = Just (Empty, input)
parseOptional input@(')':_) = Just (Empty, input)
parseOptional input         = parseTree input
```

编码直接跟随树结构，解析器则显式返回尚未消费的字符串。

### 方法二：差异列表 + `ReadP`

```haskell
import Text.ParserCombinators.ReadP

treeToStringDL :: Tree Char -> String
treeToStringDL tree = build tree ""
  where
    build Empty = id
    build (Branch c Empty Empty) = (c :)
    build (Branch c left right) =
      (c :) . ('(' :) . build left . (',' :) . build right . (')' :)

stringToTreeReadP :: String -> Maybe (Tree Char)
stringToTreeReadP input = case
  [tree | (tree, "") <- readP_to_S (treeParser <* eof) input] of
    []    -> Nothing
    trees -> Just (last trees)

treeParser :: ReadP (Tree Char)
treeParser = do
  c <- satisfy (`notElem` "(),")
  (do
      char '('
      left <- optionalTree ','
      char ','
      right <- optionalTree ')'
      char ')'
      pure (Branch c left right))
    <++ pure (leaf c)

optionalTree :: Char -> ReadP (Tree Char)
optionalTree delimiter = do
  remaining <- look
  if not (null remaining) && head remaining == delimiter
    then pure Empty
    else treeParser
```

差异列表版本通过函数组合构造字符串，不反复复制左侧前缀；`ReadP` 版本则把分隔符、选择和输入结束检查交给标准解析组合子。

## 方法对比

| 方法 | 编码 | 解码 |
|------|------|------|
| 手写递归 | 直接使用 `++` | 显式传递剩余输入 |
| 差异列表 + `ReadP` | 函数组合 | 标准解析组合子 |

## 测试

```haskell
>>> treeToString tree1
"a(b(d,e),c(,f(g,)))"
>>> stringToTree (treeToString tree1) == Just tree1
True
>>> stringToTree "a(b,c"
Nothing
>>> treeToStringDL tree1 == treeToString tree1
True
>>> stringToTreeReadP (treeToString tree1) == Just tree1
True
```

解析函数同时返回未消费后缀，顶层只接受后缀为空的结果，从而拒绝尾随垃圾字符。
