# P67 - 二叉树的括号字符串表示

> A string representation of binary trees

## 编码规则

叶子只写字符；非叶节点写成 `根(左,右)`；空子树写为空串。例如 `a(b(d,e),c(,f(g,)))`。

## 实现

```haskell
treeToString :: Tree Char -> String
treeToString Empty = ""
treeToString (Branch c Empty Empty) = [c]
treeToString (Branch c left right) =
  c : '(' : treeToString left ++ "," ++ treeToString right ++ ")"

stringToTree :: String -> Maybe (Tree Char)
stringToTree "" = Just Empty
stringToTree input = do
  (tree, rest) <- parseTree input
  if null rest then Just tree else Nothing

parseTree :: String -> Maybe (Tree Char, String)
parseTree [] = Nothing
parseTree (c:'(':rest) = do
  (left, afterLeft) <- parseOptional rest
  (right, afterRight) <- case afterLeft of
    ',':xs -> parseOptional xs
    _      -> Nothing
  case afterRight of
    ')':xs -> Just (Branch c left right, xs)
    _      -> Nothing
parseTree (c:rest) = Just (leaf c, rest)

parseOptional :: String -> Maybe (Tree Char, String)
parseOptional input@(',':_) = Just (Empty, input)
parseOptional input@(')':_) = Just (Empty, input)
parseOptional input         = parseTree input
```

## 测试

```haskell
>>> treeToString tree1
"a(b(d,e),c(,f(g,)))"
>>> stringToTree (treeToString tree1) == Just tree1
True
>>> stringToTree "a(b,c"
Nothing
```

解析函数同时返回未消费后缀，顶层只接受后缀为空的结果，从而拒绝尾随垃圾字符。
