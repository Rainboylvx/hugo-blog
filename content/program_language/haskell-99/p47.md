## 电路表示

`(-1)`、`(-2)` 分别代表两个输入。列表中第 $i$ 对 `(a,b)` 表示第 $i$ 个 NAND 门读取编号 `a`、`b`，最后一个门就是电路输出。

```haskell
evaluateCircuit :: [(Int, Int)] -> Bool -> Bool -> Bool
evaluateCircuit gates x y = last outputs
  where
    inputs = [(-1, x), (-2, y)]
    outputs = build 1 inputs gates
    build _ env [] = map snd (drop 2 (reverse env))
    build i env ((a,b):gs) =
      let value = not (lookupValue a env && lookupValue b env)
      in build (i + 1) ((i,value):env) gs
    lookupValue k env = case lookup k env of
      Just v  -> v
      Nothing -> error "evaluateCircuit: invalid gate reference"
```

## 从真值表构造电路

NAND 可以实现非、与、或。下面先构造四个最小项，再把函数为真的项析取；最后加两次非，使输出一定是最后一个门。

```haskell
data Builder = Builder Int [(Int, Int)]

emit :: Int -> Int -> Builder -> (Int, Builder)
emit a b (Builder next gs) = (next, Builder (next + 1) (gs ++ [(a,b)]))

notGate a st = emit a a st

andGate a b st0 =
  let (t, st1) = emit a b st0
  in notGate t st1

orGate a b st0 =
  let (na, st1) = notGate a st0
      (nb, st2) = notGate b st1
  in emit na nb st2

buildCircuit :: (Bool -> Bool -> Bool) -> [(Int, Int)]
buildCircuit f = gates finalState
  where
    st0 = Builder 1 []
    (nx, st1) = notGate (-1) st0
    (ny, st2) = notGate (-2) st1
    (m00, st3) = andGate nx ny st2
    (m01, st4) = andGate nx (-2) st3
    (m10, st5) = andGate (-1) ny st4
    (m11, st6) = andGate (-1) (-2) st5
    selected = [m | ((a,b),m) <- zip assignments [m00,m01,m10,m11], f a b]
    assignments = [(False,False),(False,True),(True,False),(True,True)]
    (raw, st7) = combine selected st6
    (negated, st8) = notGate raw st7
    (_, finalState) = notGate negated st8

    combine [] st = let (t, st') = emit (-1) nx st in notGate t st'
    combine (g:gs) st = foldl step (g, st) gs
    step (acc, st) g = orGate acc g st

    gates (Builder _ gs) = gs
```

## 测试

```haskell
>>> let c = buildCircuit xor'
>>> [(a,b,evaluateCircuit c a b) | a <- [False,True], b <- [False,True]]
[(False,False,False),(False,True,True),(True,False,True),(True,True,False)]
```
