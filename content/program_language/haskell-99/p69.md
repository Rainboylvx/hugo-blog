# P69 - 二叉树的点串表示

> Dotstring representation of binary trees
>
> 官方模块：`Problems.P69`　核心函数：`treeToDotstring`、`dotstringToTree`

## 编码规则

按前序遍历输出节点字符，空树输出 `.`。因为每个非空节点必有两个子树，这种编码不需要括号和分隔符也能唯一解析。

## 实现

### 方法一：递归下降

```haskell
treeToDotstring :: Tree Char -> String
treeToDotstring Empty = "."
treeToDotstring (Branch x left right) =
  x : treeToDotstring left ++ treeToDotstring right

dotstringToTree :: String -> Maybe (Tree Char)
dotstringToTree input = do
  (tree, rest) <- parseDotstring input
  if null rest then Just tree else Nothing

parseDotstring :: String -> Maybe (Tree Char, String)
parseDotstring [] = Nothing
parseDotstring ('.':rest) = Just (Empty, rest)
parseDotstring (x:rest) = do
  (left, afterLeft)   <- parseDotstring rest
  (right, afterRight) <- parseDotstring afterLeft
  pure (Branch x left right, afterRight)
```

### 方法二：反向扫描栈

```haskell
import Control.Monad (foldM)

dotstringToTreeStack :: String -> Maybe (Tree Char)
dotstringToTreeStack input = do
  stack <- foldM step [] (reverse input)
  case stack of
    [tree] -> Just tree
    _      -> Nothing
  where
    step stack '.' = Just (Empty : stack)
    step (left:right:rest) value =
      Just (Branch value left right : rest)
    step _ _ = Nothing

treeToDotstringDL :: Tree Char -> String
treeToDotstringDL tree = build tree ""
  where
    build Empty = ('.' :)
    build (Branch value left right) =
      (value :) . build left . build right
```

把点串反向读取时，两个子树一定先于根出现。`.` 把空树压栈，普通字符弹出左右两棵子树并组成新树；最终栈必须恰好只剩一棵树。

## 方法对比

| 方法 | 解码状态 | 编码构造 |
|------|----------|----------|
| 递归下降 | 未消费字符串 | `++` 拼接 |
| 反向扫描 | 显式树栈 | 差异列表 |

## 测试

```haskell
>>> treeToDotstring tree1
"abd..e..c.fg..."
>>> dotstringToTree (treeToDotstring tree1) == Just tree1
True
>>> dotstringToTree "a."
Nothing
>>> dotstringToTreeStack (treeToDotstring tree1) == Just tree1
True
>>> treeToDotstringDL tree1 == treeToDotstring tree1
True
```

编码长度恰为 $2n+1$：$n$ 个节点字符，加上 $n+1$ 个空子树标记。编码和解析都是 $O(n)$。
