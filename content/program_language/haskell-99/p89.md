# P89 - 判断二分图

> Determine whether a graph is bipartite
>
> 官方模块：`Problems.P89`　核心函数：`bipartite`

---

[← P88 连通分量](./p88) | [P90 N 皇后 →](./p90)

---

## 核心思想

二分图等价于可以用两种颜色给顶点着色，使每条边两端颜色不同。对每个连通分量做 BFS；访问边 `(u,v)` 时，未着色的 `v` 获得 `u` 的相反颜色，已着色则检查冲突。

## 实现

```haskell
bipartite :: G -> Bool
bipartite graph = colorComponents allVertices Map.empty
  where
    allVertices = fst (canonical graph)

    colorComponents remaining colors
      | Set.null remaining = True
      | otherwise = case bfs [start] (Map.insert start False colors) of
          Nothing      -> False
          Just colors' -> colorComponents
            (Set.difference remaining (Map.keysSet colors')) colors'
      where start = Set.findMin remaining

    bfs [] colors = Just colors
    bfs (u:queue) colors = process (Set.toList (neighbors u graph)) queue colors
      where
        expected = not (colors Map.! u)
        process [] rest current = bfs rest current
        process (v:vs) rest current = case Map.lookup v current of
          Just color | color /= expected -> Nothing
                     | otherwise -> process vs rest current
          Nothing -> process vs (rest ++ [v]) (Map.insert v expected current)
```

## 测试

```haskell
>>> bipartite (toG (Paths [[1,2,3,4],[1,4,5,2]]))
True
>>> bipartite (toG (Paths [[1,2,3,4],[1,4,5,2],[1,3]]))
False
```

这里用列表尾部追加模拟队列，严谨的 $O(V+E)$ 实现可换用 `Data.Sequence`。
