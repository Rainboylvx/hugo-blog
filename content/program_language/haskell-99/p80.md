# P80 - 图的表示转换

## 四种表示

```haskell
import Data.List (nub)
import qualified Data.Map.Strict as Map
import qualified Data.Set as Set

type Vertex = Int
type Edge = (Vertex, Vertex)
newtype Lists = Lists ([Vertex], [Edge]) deriving (Eq, Show)
newtype Adjacency = Adjacency [(Vertex, [Vertex])] deriving (Eq, Show)
newtype Paths = Paths [[Vertex]] deriving (Eq, Show)
newtype G = G (Map.Map Vertex (Set.Set Vertex)) deriving (Eq, Show)

normalize (u,v) | u <= v = (u,v)
                | otherwise = (v,u)
```

将每种表示先转成规范的“顶点集合 + 无向边集合”，再从规范形式构造目标类型。

## 转为规范形式

```haskell
class ConvertibleGraph g where
  canonical :: g -> (Set.Set Vertex, Set.Set Edge)

instance ConvertibleGraph Lists where
  canonical (Lists (vs,es)) =
    (Set.fromList vs, Set.fromList (map normalize es))

instance ConvertibleGraph Adjacency where
  canonical (Adjacency rows) =
    (Set.fromList (map fst rows),
     Set.fromList [normalize (u,v) | (u,ns) <- rows, v <- ns])

instance ConvertibleGraph Paths where
  canonical (Paths ps) =
    (Set.fromList (concat ps),
     Set.fromList [normalize e | path <- ps, e <- zip path (drop 1 path)])

instance ConvertibleGraph G where
  canonical (G graph) =
    (Map.keysSet graph,
     Set.fromList [normalize (u,v) | (u,ns) <- Map.toList graph
                                   , v <- Set.toList ns])
```

## 构造目标表示

```haskell
toLists graph = Lists (Set.toList vs, Set.toList es)
  where (vs,es) = canonical graph

toAdjacency graph = Adjacency
  [(v, [other v e | e <- Set.toList es, contains v e]) | v <- Set.toList vs]
  where
    (vs,es) = canonical graph
    contains v (a,b) = v == a || v == b
    other v (a,b) = if v == a then b else a

toPaths graph = Paths
  (map (\(u,v) -> [u,v]) (Set.toList es) ++ map (:[]) isolated)
  where
    (vs,es) = canonical graph
    touched = Set.fromList [x | (u,v) <- Set.toList es, x <- [u,v]]
    isolated = Set.toList (Set.difference vs touched)

toG graph = G $ Set.foldl addEdge base es
  where
    (vs,es) = canonical graph
    base = Map.fromSet (const Set.empty) vs
    addEdge m (u,v) = Map.insertWith Set.union u (Set.singleton v)
                    $ Map.insertWith Set.union v (Set.singleton u) m
```

## 测试

```haskell
>>> canonical (toG (Paths [[1,2,3],[1,3],[4]])) ==
... canonical (Lists ([1,2,3,4],[(1,2),(2,3),(1,3)]))
True
```
