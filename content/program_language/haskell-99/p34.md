# P34 - 欧拉函数

> Calculate Euler's totient function
>
> 官方模块：`Problems.P34`　核心函数：`totient`

## 题目描述

欧拉函数 $\varphi(m)$ 表示 $1\le r\le m$ 中与 $m$ 互质的整数个数。由于 $m>1$ 时 $m$ 不与自身互质，使用 `[1..m]` 或 `[1..m-1]` 结果相同。

## 函数签名与实现

```haskell
totient :: Integral a => a -> a
totient m
  | m < 1     = error "totient: positive input required"
  | otherwise = fromIntegral . length $ filter (coprime m) [1..m]
```

这是定义的直接翻译：筛选互质整数，再计数。P37 会用欧拉乘积公式将它优化。

## 测试

```haskell
>>> totient 1
1
>>> totient 10
4
>>> totient 315
144
```

因为 1、3、7、9 与 10 互质，所以 $\varphi(10)=4$。

## 复杂度

执行 $m$ 次最大公约数计算，粗略时间复杂度为 $O(m\log m)$。
