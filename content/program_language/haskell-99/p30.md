# P30 - 矩阵快速幂计算斐波那契数

> Fibonacci numbers with matrix exponentiation
>
> 官方模块：`Problems.P30`　核心函数：`fibonacci'`

---

[← P29 斐波那契数](./p29) | [P31 判断素数 →](./p31)

## 核心公式

利用转移矩阵

$$
\begin{pmatrix} F(n+1) \\ F(n) \end{pmatrix} = \begin{pmatrix} 1 & 1 \\ 1 & 0 \end{pmatrix}^{n-1} \begin{pmatrix} 1 \\ 0 \end{pmatrix}
$$

问题变成整数矩阵的快速幂。

## 函数签名

```haskell
fibonacci' :: Integral a => a -> a
```

## 实现

### 方法一：矩阵快速幂（O(log n)）

```haskell
type Matrix a = ((a, a), (a, a))

fibonacci' :: Integral a => a -> a
fibonacci' n
  | n < 1     = error "fibonacci': index must be positive"
  | otherwise = fst (apply (power q (n - 1)) (1, 0))
  where q = ((1, 1), (1, 0))

apply :: Num a => Matrix a -> (a, a) -> (a, a)
apply ((a,b),(c,d)) (x,y) = (a*x + b*y, c*x + d*y)

multiply :: Num a => Matrix a -> Matrix a -> Matrix a
multiply ((a,b),(c,d)) ((e,f),(g,h)) =
  ((a*e+b*g, a*f+b*h), (c*e+d*g, c*f+d*h))

identity :: Num a => Matrix a
identity = ((1,0),(0,1))

power :: Num a => Matrix a -> a -> Matrix a
power _ 0 = identity
power m n
  | even n    = power (multiply m m) (n `div` 2)
  | otherwise = multiply m (power m (n - 1))
```

复杂度 O(log n)，适合计算超大索引（如 `fibonacci' 10^6`）。矩阵乘法用元组表示，清晰但代码量大。

#### C++ 对照实现（Codeforces 风格）

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

// 2x2 矩阵，a[i][j] 表示第 i 行第 j 列
// 表示 | a[0][0]  a[0][1] |
//      | a[1][0]  a[1][1] |

// 矩阵乘法：C = A * B
// C[i][j] = sum_k A[i][k] * B[k][j]
void mul(ll A[2][2], ll B[2][2], ll C[2][2]) {
    ll tmp[2][2] = {{0, 0}, {0, 0}};
    for (int i = 0; i < 2; i++) {
        for (int j = 0; j < 2; j++) {
            for (int k = 0; k < 2; k++) {
                tmp[i][j] += A[i][k] * B[k][j];
            }
        }
    }
    // 结果写回 C
    for (int i = 0; i < 2; i++) {
        for (int j = 0; j < 2; j++) {
            C[i][j] = tmp[i][j];
        }
    }
}

// 矩阵快速幂：把 A 变成 A^n，结果写在 res 里
// 原理：二进制拆分 n
//   若 n 的某一位是 1，就把当前的 A^{2^i} 乘进答案
//   然后 A 自身平方：A -> A^2 -> A^4 -> ...
void qpow(ll A[2][2], ll n, ll res[2][2]) {
    // 单位矩阵 I：乘任何矩阵都不变
    res[0][0] = 1; res[0][1] = 0;
    res[1][0] = 0; res[1][1] = 1;

    ll base[2][2];
    for (int i = 0; i < 2; i++) {
        for (int j = 0; j < 2; j++) {
            base[i][j] = A[i][j];
        }
    }

    while (n > 0) {
        if (n & 1) {
            // n 当前最低位是 1：res = res * base
            ll tmp[2][2];
            mul(res, base, tmp);
            for (int i = 0; i < 2; i++) {
                for (int j = 0; j < 2; j++) {
                    res[i][j] = tmp[i][j];
                }
            }
        }
        // base = base * base
        ll tmp[2][2];
        mul(base, base, tmp);
        for (int i = 0; i < 2; i++) {
            for (int j = 0; j < 2; j++) {
                base[i][j] = tmp[i][j];
            }
        }
        n >>= 1;  // 处理下一位
    }
}

// 用矩阵快速幂求第 n 个斐波那契数
// 转移：
// | F(n+1) |   | 1 1 |^{n-1}   | F(1) |     | F(1)=1 |
// | F(n)   | = | 1 0 |       * | F(0) | ,   | F(0)=0 |
//
// 所以 F(n) = (转移矩阵)^{n-1} 的左上角 a[0][0]
ll fib(ll n) {
    if (n < 1) return 0;  // 题目要求正索引；按需改

    // 转移矩阵 Q
    ll Q[2][2] = {{1, 1}, {1, 0}};
    ll P[2][2];
    qpow(Q, n - 1, P);

    // P * (1, 0)^T = (P[0][0], P[1][0])
    // 其中 P[0][0] = F(n), P[1][0] = F(n-1)
    return P[0][0];
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    ll n;
    cin >> n;
    cout << fib(n) << '\n';
    return 0;
}
```

| Haskell | C++ |
|---------|-----|
| `Matrix a = ((a,a),(a,a))` | `ll A[2][2]` 普通二维数组 |
| `multiply` | `mul` |
| `power` | `qpow`（二进制快速幂） |
| `apply` 后取 `fst` | `P[0][0]` 即 `F(n)` |

说明：上面没取模。CF 上通常要 `mod` 的话，在 `mul` 里每次加法后 `% MOD` 即可。

### 方法二：使用线性递推的 fast doubling（推荐）

```haskell
fibonacci' :: Integral a => a -> a
fibonacci' n
  | n < 1     = error "fibonacci': index must be positive"
  | otherwise = fst (fibPair n)
  where
    fibPair 0 = (0, 1)
    fibPair k =
      let (a, b) = fibPair (k `div` 2)
          c = a * (b * 2 - a)
          d = a * a + b * b
      in if even k then (c, d) else (d, c + d)
```

Fast doubling 公式：
- $F(2k) = F(k) \times (2F(k+1) - F(k))$
- $F(2k+1) = F(k+1)^2 + F(k)^2$

同样 O(log n)，但无需矩阵，代码更紧凑，实际常数也更小。

### 方法三：调用 P29 的尾递归（朴素 O(n)）

```haskell
fibonacci' :: Integral a => a -> a
fibonacci' n = fibonacci n   -- 复用 P29 的尾递归实现
```

对于 n 不大（< 10^6）的场景，O(n) 尾递归已经足够，代码最简单。

## 方法对比

| 方法 | 复杂度 | 代码量 | 适用场景 |
|------|--------|--------|----------|
| 矩阵快速幂 | O(log n) | ~20 行 | 教学展示矩阵运算 |
| Fast doubling | O(log n) | ~8 行 | 生产推荐，简洁高效 |
| P29 尾递归 | O(n) | 1 行 | n 较小时最直接 |
