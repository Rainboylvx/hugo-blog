# P61 - 收集叶子节点与内部节点

> Collect nodes of a binary tree
>
> 官方模块：`Problems.P61`　核心函数：`leaves`、`internals`

## 题目描述

叶子节点没有任何子节点；内部节点至少有一个非空子树。分别按从左到右的顺序收集二者。

## 实现

```haskell
leaves :: Tree a -> [a]
leaves Empty                  = []
leaves (Branch x Empty Empty) = [x]
leaves (Branch _ left right)  = leaves left ++ leaves right

internals :: Tree a -> [a]
internals Empty                  = []
internals (Branch _ Empty Empty) = []
internals (Branch x left right)  =
  internals left ++ [x] ++ internals right
```

`internals` 采用中序顺序；核心要求是每个非叶节点恰好出现一次。

## 测试

```haskell
>>> leaves tree1
"deg"
>>> internals tree1
"bacf"
>>> leaves (leaf 42)
[42]
>>> internals (leaf 42)
[]
```

由于反复使用 `++`，极端偏斜树上可能产生二次开销；可用差异列表或累加参数优化。
