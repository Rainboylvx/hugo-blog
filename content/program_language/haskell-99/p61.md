# P61 - 收集叶子节点与内部节点

> Collect nodes of a binary tree
>
> 官方模块：`Problems.P61`　核心函数：`leaves`、`internals`

---

## 题目描述

叶子节点没有任何子节点；内部节点至少有一个非空子树。分别按从左到右的顺序收集二者。

## 函数签名

```haskell
leaves :: Tree a -> [a]
internals :: Tree a -> [a]
```

## 实现

### 方法一：直接递归

```haskell
leaves :: Tree a -> [a]
leaves Empty                  = []
leaves (Branch x Empty Empty) = [x]
leaves (Branch _ left right)  = leaves left ++ leaves right

internals :: Tree a -> [a]
internals Empty                  = []
internals (Branch _ Empty Empty) = []
internals (Branch x left right)  =
  internals left ++ [x] ++ internals right
```

叶子按左右子树的顺序拼接。内部节点采用中序顺序：左子树、当前节点、右子树。

### 方法二：累加参数（差异列表）

```haskell
leavesAcc :: Tree a -> [a]
leavesAcc tree = gather tree []
  where
    gather Empty tail = tail
    gather (Branch x Empty Empty) tail = x : tail
    gather (Branch _ left right) tail =
      gather left (gather right tail)

internalsAcc :: Tree a -> [a]
internalsAcc tree = gather tree []
  where
    gather Empty tail = tail
    gather (Branch _ Empty Empty) tail = tail
    gather (Branch x left right) tail =
      gather left (x : gather right tail)
```

`gather tree tail` 表示“遍历结果后面继续接 `tail`”，因此不需要第三方 DList 包，也避免反复执行 `++`。

## 方法对比

| 方法 | 特点 |
|------|------|
| 直接递归 | 与节点定义对应，最容易阅读 |
| 累加参数 | 单次线性构造结果，适合偏斜树 |

## 测试

```haskell
>>> leaves (Branch 'a' (leaf 'b') (leaf 'c'))
"bc"
>>> internals (Branch 'a' (leaf 'b') (leaf 'c'))
"a"
>>> leavesAcc (Branch 'a' (leaf 'b') (leaf 'c')) == leaves (Branch 'a' (leaf 'b') (leaf 'c'))
True
>>> internalsAcc (Branch 'a' (leaf 'b') (leaf 'c')) == internals (Branch 'a' (leaf 'b') (leaf 'c'))
True
```
