# P61 - 收集叶子节点

> Collect the leaves of a binary tree
>
> 官方模块：`Problems.P61`　核心函数：`leaves`

---

[← P60 指定节点数高度平衡树](./p60) | [P62 收集指定层节点 →](./p62)

---

## 函数签名

```haskell
leaves :: Tree a -> [a]
```

## 实现

### 方法一：递归（左 + 右）

```haskell
leaves :: Tree a -> [a]
leaves Empty                  = []
leaves (Branch x Empty Empty) = [x]
leaves (Branch _ left right)  = leaves left ++ leaves right
```

叶子节点定义为左右子树都为空的分支。非叶子节点递归收集左右子树的叶子。

### 方法二：用 concatMap

```haskell
leaves :: Tree a -> [a]
leaves Empty = []
leaves (Branch x Empty Empty) = [x]
leaves (Branch _ l r) = concatMap leaves [l, r]
```

### 方法三：用 DList 优化

```haskell
import Data.DList (DList, toList, singleton, append)

leaves :: Tree a -> [a]
leaves = toList . go
  where
    go Empty = mempty
    go (Branch x Empty Empty) = singleton x
    go (Branch _ l r) = go l `mappend` go r
```

用 `DList` 避免 `++` 的左嵌套带来的 O(n²) 问题。

## 测试

```haskell
>>> leaves (Branch 'a' (leaf 'b') (leaf 'c'))
"bc"
>>> leaves Empty
[]
```
