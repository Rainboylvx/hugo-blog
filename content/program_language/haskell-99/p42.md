# P42 - 模乘逆元

> Compute a modular multiplicative inverse
>
> 官方模块：`Problems.P42`　核心函数：`multiplicativeInverse`

## 题目描述

求 $0\le x<n$，使 $ax\equiv1\pmod n$。逆元存在当且仅当 $\gcd(a,n)=1$，因此返回类型使用 `Maybe`。

## 实现

扩展欧几里得算法同时求出 $g,s,t$，满足 $as+nt=g$。当 $g=1$ 时，$s$ 就是逆元。

```haskell
multiplicativeInverse :: Integral a => a -> a -> Maybe a
multiplicativeInverse a n
  | n <= 1    = Nothing
  | g == 1    = Just (x `mod` n)
  | otherwise = Nothing
  where
    (g, x, _) = extendedGCD (a `mod` n) n

extendedGCD :: Integral a => a -> a -> (a, a, a)
extendedGCD 0 b = (abs b, 0, signum b)
extendedGCD a b =
  let (g, x, y) = extendedGCD (b `mod` a) a
  in (g, y - (b `div` a) * x, x)
```

## 测试

```haskell
>>> multiplicativeInverse 3 5
Just 2
>>> multiplicativeInverse 48 127
Just 45
>>> multiplicativeInverse 824 93
Just 50
>>> multiplicativeInverse 48 93
Nothing
```

## 复杂度

与欧几里得算法相同，时间复杂度为 $O(\log n)$。
