# P42 - 模乘逆元

> Determine the modular multiplicative inverse
>
> 官方模块：`Problems.P42`　核心函数：`multiplicativeInverse`

---

[← P41 哥德巴赫数对列表](./p41) | [P43 高斯整数整除 →](./p43)

---

## 题目描述

求模 $n$ 下 $a$ 的乘法逆元 $x$，满足 $ax \equiv 1 \pmod n$。逆元存在当且仅当 $\gcd(a, n) = 1$。

## 函数签名

```haskell
multiplicativeInverse :: Integral a => a -> a -> Maybe a
```

## 实现

### 方法一：扩展欧几里得算法

```haskell
multiplicativeInverse :: Integral a => a -> a -> Maybe a
multiplicativeInverse a n
  | n <= 1    = Nothing
  | g == 1    = Just (x `mod` n)
  | otherwise = Nothing
  where
    (g, x, _) = extendedGCD (a `mod` n) n

extendedGCD :: Integral a => a -> a -> (a, a, a)
extendedGCD 0 b = (abs b, 0, signum b)
extendedGCD a b =
  let (g, x, y) = extendedGCD (b `mod` a) a
  in (g, y - (b `div` a) * x, x)
```

扩展欧几里得算法同时求出 $g,s,t$，满足 $as+nt=g$。当 $g=1$ 时，$s$ 就是逆元。

### 方法二：迭代版扩展欧几里得

```haskell
multiplicativeInverse :: Integral a => a -> a -> Maybe a
multiplicativeInverse a n
  | n <= 1    = Nothing
  | g == 1    = Just (x `mod` n)
  | otherwise = Nothing
  where
    (g, x, _) = iter 0 1 n (a `mod` n)
    iter s0 s1 r0 r1
      | r1 == 0   = (r0, s0, (r0 - s0 * (a `mod` n)) `div` n)
      | otherwise = let q = r0 `div` r1
                    in iter s1 (s0 - q * s1) r1 (r0 - q * r1)
```

非递归版本，避免在大数上递归栈溢出。

### 方法三：朴素枚举（仅用于验证）

```haskell
multiplicativeInverse :: Integral a => a -> a -> Maybe a
multiplicativeInverse a n
  | n <= 1    = Nothing
  | otherwise = listToMaybe [x | x <- [1..n-1], (a * x) `mod` n == 1]
```

从定义出发，枚举 $1..n-1$ 检查乘积模 n 是否为 1。小 n 可用，大 n 极慢。

## 方法对比

| 方法 | 复杂度 | 特点 |
|------|--------|------|
| 递归扩展 GCD | O(log n) | 标准做法，推荐 |
| 迭代扩展 GCD | O(log n) | 无栈溢出风险 |
| 朴素枚举 | O(n) | 教学验证用 |

## 测试

```haskell
>>> multiplicativeInverse 3 7
Just 5      -- 因为 3 * 5 = 15 ≡ 1 (mod 7)
>>> multiplicativeInverse 2 6
Nothing     -- gcd(2,6) = 2 ≠ 1
>>> multiplicativeInverse 1 2
Just 1
```
