# P70 - 多路树的节点串表示

> Construct a multiway tree from a node string
>
> 官方模块：`Problems.P70`　核心函数：`stringToMultitree`、`multitreeToString`

## 数据结构与编码

多路树不能为空，每个节点保存任意数量的子树。深度优先输出节点字符，每次回退到父节点时输出 `^`，包括从根退出。

```haskell
data MultiwayTree a = MultiwayTree a [MultiwayTree a]
  deriving (Eq, Show)

multitreeToString :: MultiwayTree Char -> String
multitreeToString (MultiwayTree x children) =
  x : concatMap multitreeToString children ++ "^"
```

## 解析

### 方法一：递归下降

```haskell
stringToMultitree :: String -> Maybe (MultiwayTree Char)
stringToMultitree input = do
  (tree, rest) <- parseNode input
  if null rest then Just tree else Nothing

parseNode :: String -> Maybe (MultiwayTree Char, String)
parseNode []       = Nothing
parseNode ('^':_)  = Nothing
parseNode (x:rest) = do
  (children, remaining) <- parseChildren rest
  pure (MultiwayTree x children, remaining)

parseChildren :: String -> Maybe ([MultiwayTree Char], String)
parseChildren [] = Nothing
parseChildren ('^':rest) = Just ([], rest)
parseChildren input = do
  (child, afterChild) <- parseNode input
  (siblings, rest)    <- parseChildren afterChild
  pure (child : siblings, rest)
```

### 方法二：显式节点栈

```haskell
import Control.Monad (foldM)

stringToMultitreeStack :: String -> Maybe (MultiwayTree Char)
stringToMultitreeStack input = do
  (frames, result) <- foldM step ([], Nothing) input
  case (frames, result) of
    ([], Just tree) -> Just tree
    _               -> Nothing
  where
    step (_, Just _) _ = Nothing
    step (frames, Nothing) '^' = close frames
    step (frames, Nothing) value =
      Just ((value, []) : frames, Nothing)

    close [] = Nothing
    close [(value, reversedChildren)] =
      Just ([], Just (MultiwayTree value (reverse reversedChildren)))
    close ((value, reversedChildren):(parent, siblings):rest) =
      let child = MultiwayTree value (reverse reversedChildren)
      in Just ((parent, child : siblings) : rest, Nothing)

multitreeToStringDL :: MultiwayTree Char -> String
multitreeToStringDL tree = build tree ""
  where
    build (MultiwayTree value children) =
      (value :) . foldr ((.) . build) id children . ('^' :)
```

普通字符把新节点框架压栈，`^` 关闭栈顶节点并把它接到父节点。根节点关闭后不得再出现字符。编码的差异列表版本同样只用函数组合追加内容。

## 方法对比

| 方法 | 解析结构 | 特点 |
|------|----------|------|
| 递归下降 | 调用栈对应树层级 | 与文法直接对应 |
| 显式节点栈 | 框架保存尚未关闭的节点 | 单次 fold，状态清晰 |

## 测试

```haskell
>>> multitreeToString multitree5
"afg^^c^bd^e^^^"
>>> stringToMultitree "afg^^c^bd^e^^^" == Just multitree5
True
>>> stringToMultitree "a^junk"
Nothing
>>> stringToMultitreeStack (multitreeToString multitree5) == Just multitree5
True
>>> multitreeToStringDL multitree5 == multitreeToString multitree5
True
```

编码和解析都线性访问字符串，时间复杂度 $O(n)$。
