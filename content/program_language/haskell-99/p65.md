# P65 - 二叉树布局：层级等距

> Binary tree layout: constant distance between nodes at each level
>
> 官方模块：`Problems.P65`　核心函数：`layoutLevelConstant`

## 题目描述

同一层的节点水平间距恒定。越靠近根的层间距越小（因为节点少），越深间距越大。

## 函数签名

```haskell
layoutLevelConstant :: Tree a -> Tree (a, (Int, Int))
```

## 实现

### 方法一：按深度位移

```haskell
layoutLevelConstant :: Tree a -> Tree (a, (Int, Int))
layoutLevelConstant tree = shift (1 - minimumX raw) raw
  where
    gap = 2 ^ max 0 (treeHeight tree - 2)
    raw = place tree 0 1 gap
    shift offset (Branch (x, (_, y)) l r) =
      Branch ((x + offset), (x + offset, y)) (shift offset l) (shift offset r)
    shift _ Empty = Empty

    place (Branch x l r) leftEdge level gap
      | null children = leaf (x, (leftEdge, level))
      | otherwise = let width = 2 ^ (treeHeight tree - level - 1) * gap
                        xpos = leftEdge + width
                    in Branch (x, (xpos, level))
                              (place l leftEdge (level + 1) gap)
                              (place r (xpos + gap) (level + 1) gap)
    place Empty _ _ _ = Empty
    children = treeSize l + treeSize r
```
