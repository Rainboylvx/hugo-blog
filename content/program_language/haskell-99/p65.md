# P65 - 二叉树布局：层级等距

> Binary tree layout: constant distance between nodes at each level
>
> 官方模块：`Problems.P65`　核心函数：`layoutLevelConstant`

---

[← P64 中序布局](./p64) | [P66 紧凑布局 →](./p66)

## 题目描述

父子节点的水平距离只由深度决定，越靠近根距离越大，向下一层距离减半。布局完成后最左节点的横坐标为 1。

## 函数签名

```haskell
layoutLevelConstant :: Tree a -> Tree (a, (Int, Int))
```

## 实现

### 方法一：按深度位移

```haskell
layoutLevelConstant :: Tree a -> Tree (a, (Int, Int))
layoutLevelConstant tree = shift (1 - minimumX raw) raw
  where
    gap = 2 ^ max 0 (treeHeight tree - 2)
    raw = place tree 0 1 gap

place :: Tree a -> Int -> Int -> Int -> Tree (a, (Int, Int))
place Empty _ _ _ = Empty
place (Branch value left right) x depth gap =
  Branch (value, (x, depth))
    (place left  (x - gap) (depth + 1) (gap `div` 2))
    (place right (x + gap) (depth + 1) (gap `div` 2))

minimumX :: Tree (a, (Int, Int)) -> Int
minimumX Empty = 0
minimumX (Branch (_, (x, _)) left right) =
  minimum [x, minimumX left, minimumX right]

shift :: Int -> Tree (a, (Int, Int)) -> Tree (a, (Int, Int))
shift _ Empty = Empty
shift amount (Branch (value, (x, y)) left right) =
  Branch (value, (x + amount, y))
    (shift amount left)
    (shift amount right)
```

先以根横坐标 0 递归布局，再根据最小横坐标整体平移。树高决定根到孩子的初始距离。

### 方法二：满二叉树槽位公式

```haskell
layoutLevelConstantByIndex :: Tree a -> Tree (a, (Int, Int))
layoutLevelConstantByIndex Empty = Empty
layoutLevelConstantByIndex tree = shiftBy (1 - minimumPosition raw) raw
  where
    height = treeHeight tree
    raw = placeAt tree 1 0

    placeAt Empty _ _ = Empty
    placeAt (Branch value left right) depth slot =
      Branch (value, (x, depth))
        (placeAt left (depth + 1) (2 * slot))
        (placeAt right (depth + 1) (2 * slot + 1))
      where
        x = (2 * slot + 1) * 2 ^ (height - depth)

    minimumPosition = minimum . positions

    positions Empty = []
    positions (Branch (_, (x, _)) left right) =
      x : positions left ++ positions right

    shiftBy _ Empty = Empty
    shiftBy amount (Branch (value, (x, y)) left right) =
      Branch (value, (x + amount, y))
        (shiftBy amount left)
        (shiftBy amount right)
```

把当前树嵌入同高度的满二叉树。深度为 `depth` 的第 `slot` 个槽位横坐标可直接由公式算出，最后再按实际存在节点的最小横坐标归一化。

## 方法对比

| 方法 | 核心思路 |
|------|----------|
| 按深度位移 | 从根坐标出发递归加减间距 |
| 槽位公式 | 由深度和层内编号直接计算坐标 |

## 测试

```haskell
>>> layoutLevelConstant (Branch 'a' (leaf 'b') (leaf 'c'))
Branch ('a',(2,1)) (Branch ('b',(1,2)) Empty Empty) (Branch ('c',(3,2)) Empty Empty)
>>> layoutLevelConstantByIndex (Branch 'a' (leaf 'b') (leaf 'c'))
Branch ('a',(2,1)) (Branch ('b',(1,2)) Empty Empty) (Branch ('c',(3,2)) Empty Empty)
```
