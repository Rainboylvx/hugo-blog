# P32 - 最大公约数

> Determine the greatest common divisor
>
> 官方模块：`Problems.P32`　核心函数：`myGCD`

---

[← P31 判断素数](./p31) | [P33 判断互质 →](./p33)

## 核心思想

欧几里得算法基于 $\gcd(a,b)=\gcd(b,a\bmod b)$。余数严格变小，所以递归必然终止。

## 函数签名与实现

```haskell
myGCD :: Integral a => a -> a -> a
myGCD a b = go (abs a) (abs b)
  where
    go x 0 = x
    go x y = go y (x `mod` y)
```

这里把 `gcd a 0` 定义为 `abs a`，并得到 `gcd 0 0 = 0`，与 Prelude 的 `gcd` 行为一致。

## 测试

```haskell
>>> myGCD 36 63
9
>>> myGCD (-36) 63
9
>>> myGCD 17 0
17
>>> myGCD 0 0
0
```

## 复杂度

欧几里得算法的递归次数为 $O(\log \min(a,b))$。
