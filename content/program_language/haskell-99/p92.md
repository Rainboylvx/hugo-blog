# P92 - 优雅树标号

> Graceful tree labeling
>
> 官方模块：`Problems.P92`　核心函数：`gracefulTree`

---

[← P91 骑士巡游](./p91) | [P93 算术谜题 →](./p93)

## 定义

含 $n$ 个顶点的树若能用 $1..n$ 双射标号，并让 $n-1$ 条边两端标号差的绝对值恰好组成 $1..n-1$，这个标号就是优雅标号。

## 实现

### 方法一：枚举完整排列

```haskell
import Data.List (delete, permutations)

gracefulTree :: G -> Maybe (Map.Map Vertex Int)
gracefulTree graph
  | not (isTree graph) = Nothing
  | otherwise = firstJust
      [Just labeling | labels <- permutations [1..n]
                     , let labeling = Map.fromList (zip vertices labels)
                     , graceful labeling]
  where
    (vs,es) = canonical graph
    vertices = Set.toList vs
    n = length vertices
    graceful labeling =
      Set.fromList [abs (labeling Map.! u - labeling Map.! v)
                   | (u,v) <- Set.toList es]
      == Set.fromList [1..n-1]

firstJust [] = Nothing
firstJust (Just value:_) = Just value
firstJust (Nothing:rest) = firstJust rest
```

这个版本直接枚举所有 $n!$ 个双射，代码短且定义清楚。实用版本应优先标记高度数顶点，并在部分标号产生重复边差时立即回溯。

### 方法二：部分标号回溯

```haskell
import Data.List (delete, sortOn)
import Data.Ord (Down(..))

gracefulTreeBacktracking :: G -> Maybe (Map.Map Vertex Int)
gracefulTreeBacktracking graph
  | not (isTree graph) = Nothing
  | otherwise = assign orderedVertices Map.empty Set.empty [1..n]
  where
    vertices = Set.toList (fst (canonical graph))
    n = length vertices
    orderedVertices = sortOn
      (Down . Set.size . (`neighbors` graph)) vertices

    assign [] labeling _ _ = Just labeling
    assign (vertex:rest) labeling usedDifferences availableLabels =
      firstJust
        [assign rest
                (Map.insert vertex label labeling)
                (Set.union usedDifferences newDifferences)
                (delete label availableLabels)
        | label <- availableLabels
        , let newDifferences = Set.fromList
                [abs (label - neighborLabel)
                | neighbor <- Set.toList (neighbors vertex graph)
                , Just neighborLabel <- [Map.lookup neighbor labeling]]
        , Set.size newDifferences == length
            [() | neighbor <- Set.toList (neighbors vertex graph)
                , Map.member neighbor labeling]
        , Set.null (Set.intersection usedDifferences newDifferences)
        ]
```

每给一个顶点标号，就计算它与已标号邻居产生的新边差。新边差内部重复或与此前边差冲突时立即回溯。优先处理高度数顶点可以更早形成约束。

## 方法对比

| 方法 | 检查时机 |
|------|----------|
| 完整排列 | n 个顶点全部标号后检查 |
| 部分标号回溯 | 每加入一个标号就检查新边差 |

## 测试

```haskell
>>> let path4 = toG (Paths [[1,2,3,4]])
>>> fmap (const True) (gracefulTree path4)
Just True
>>> gracefulTree (toG (Paths [[1,2,3,1]]))
Nothing
>>> fmap (const True) (gracefulTreeBacktracking path4)
Just True
```

不同枚举顺序可能返回不同但同样正确的标号，测试应验证标号集合和边差集合，而不是固定某一映射。
