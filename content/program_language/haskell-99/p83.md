# P83 - 构造所有生成树

> Construct all spanning trees
>
> 官方模块：`Problems.P83`　核心函数：`spanningTrees`、`isTree`、`isConnected`

## 判定基础

含 $n$ 个顶点的无向图是树，当且仅当它连通且恰有 $n-1$ 条边。因此生成树可从原图边集中选择 $n-1$ 条，再过滤连通候选。

### 方法一：组合后过滤

```haskell
isConnected :: G -> Bool
isConnected graph
  | Set.null vs = True
  | otherwise   = reachable (Set.findMin vs) == vs
  where
    (vs,_) = canonical graph
    reachable start = visit Set.empty [start]
    visit seen [] = seen
    visit seen (v:stack)
      | Set.member v seen = visit seen stack
      | otherwise = visit (Set.insert v seen)
          (Set.toList (neighbors v graph) ++ stack)

isTree :: G -> Bool
isTree graph = isConnected graph && Set.size es == max 0 (Set.size vs - 1)
  where (vs,es) = canonical graph

spanningTrees :: G -> [G]
spanningTrees graph
  | Set.null vs = []
  | otherwise =
      [tree | chosen <- combinations (Set.size vs - 1) (Set.toList es)
            , let tree = toG (Lists (Set.toList vs, chosen))
            , isConnected tree]
  where (vs,es) = canonical graph
```

### 方法二：增量合并连通分量

```haskell
spanningTreesIncremental :: G -> [G]
spanningTreesIncremental graph
  | Set.null vertices = []
  | otherwise =
      [toG (Lists (Set.toList vertices, chosen))
      | chosen <- grow (Set.toList edges) initialComponents []]
  where
    (vertices, edges) = canonical graph
    target = Set.size vertices - 1
    initialComponents = map Set.singleton (Set.toList vertices)

    grow remaining components chosen
      | length chosen == target =
          [reverse chosen | length components == 1]
      | length chosen + length remaining < target = []
    grow [] _ _ = []
    grow (edge@(u, v):rest) components chosen =
      grow rest components chosen ++ include
      where
        componentU = componentOf u components
        componentV = componentOf v components
        include
          | componentU == componentV = []
          | otherwise = grow rest merged (edge : chosen)
        merged = Set.union componentU componentV
               : filter (\component -> component /= componentU && component /= componentV)
                        components

    componentOf vertex = head . filter (Set.member vertex)
```

每加入一条边就合并两个连通分量；若端点已经在同一分量，加入它一定成环，可以立刻剪枝。选满 $n-1$ 条边时只需检查是否剩一个分量，不必重新执行 DFS。

## 方法对比

| 方法 | 候选检查 |
|------|----------|
| 组合后过滤 | 先选满 n-1 条边，再检查连通性 |
| 增量合并 | 选边时立即排除环和边数不足分支 |

## 测试

```haskell
>>> isTree (toG (Paths [[1,2,3],[1,4,5]]))
True
>>> isConnected (toG (Lists ([1,2,3],[])))
False
>>> length (spanningTrees (toG (Paths [[1,2,3,1]])))
3
>>> length (spanningTreesIncremental (toG (Paths [[1,2,3,1]])))
3
```

组合枚举最坏需要检查 $\binom{m}{n-1}$ 个候选，适合练习而非大型图。
