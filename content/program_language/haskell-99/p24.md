# P24 - 从 1 到 M 随机抽取 N 个不同整数

> Draw N different random numbers from 1 to M
>
> 官方模块：`Problems.P24`　核心函数：`randomDraw`

## 题目描述

从集合 $\{1,2,\ldots,m\}$ 中随机抽取 $n$ 个互不相同的整数。这就是 P23 在整数范围上的直接应用。

## 函数签名

```haskell
import System.Random (RandomGen)

randomDraw :: RandomGen g => Int -> Int -> g -> ([Int], g)
```

第一个参数是抽取数量 $n$，第二个参数是上界 $m$。

## 实现

复用 P23 的 `randomSelect`：

```haskell
import System.Random (RandomGen)

randomDraw :: RandomGen g => Int -> Int -> g -> ([Int], g)
randomDraw n m = randomSelect [1 .. max 0 m] n
```

部分应用后，`randomDraw n m` 仍然是一个等待随机数生成器的函数。

## 测试

```haskell
import Data.List (nub)
import System.Random (mkStdGen)

>>> let (xs, _) = randomDraw 6 49 (mkStdGen 111)
>>> length xs == 6
True
>>> length (nub xs) == 6 && all (`elem` [1..49]) xs
True
>>> length . fst $ randomDraw 20 5 (mkStdGen 1)
5
```

## 边界条件

- $n \le 0$ 时返回空列表。
- $n > m$ 时最多返回 $m$ 个数，而不是重复抽取。
- $m \le 0$ 时候选集合为空。

## 要点总结

函数式编程中，小函数的组合往往比重新实现算法更可靠。P24 的全部随机性、去重和生成器传递都已经由 P23 解决。
