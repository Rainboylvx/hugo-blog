# P24 - 从 1..M 中随机选 N 个不同数

> Draw N different numbers from the set {1 .. M}
>
> 官方模块：`Problems.P24`　核心函数：`randomDraw`

---

[← P23 从列表随机选 N 个元素](./p23) | [P25 随机排列列表 →](./p25)

---

## 题目描述

从 $1$ 到 $M$ 的整数中随机选出 $N$ 个不同的数。这是 P23 的直接应用——把 `[1..M]` 作为候选列表。

## 函数签名

```haskell
import System.Random (RandomGen)

randomDraw :: RandomGen g => Int -> Int -> g -> ([Int], g)
```

参数顺序：`randomDraw n m gen`（选 N 个，范围 1..M）。

## 实现

### 方法一：基于 randomSelect

```haskell
import System.Random (RandomGen)

randomDraw :: RandomGen g => Int -> Int -> g -> ([Int], g)
randomDraw n m = randomSelect [1 .. max 0 m] n
```

如果 M ≤ 0，`[1 .. max 0 m]` 得到 `[]`，`randomSelect` 返回空列表。一行搞定。

### 方法二：直接实现——抽取不放回

```haskell
import System.Random (RandomGen, randomR)

randomDraw :: RandomGen g => Int -> Int -> g -> ([Int], g)
randomDraw n m gen = go count [1 .. max 0 m] gen []
  where
    count = max 0 (min n m)
    go 0 _     g acc = (acc, g)
    go _ []    g acc = (acc, g)
    go k pool g acc =
      let (i, g') = randomR (0, length pool - 1) g
          x       = pool !! i
          pool'   = take i pool ++ drop (i + 1) pool
      in go (k - 1) pool' g' (x : acc)
```

不依赖 `randomSelect`，从数组中用下标随机取元素并移除，逻辑自包含。

### 方法三：——洗牌后取前 N 个

```haskell
import System.Random (RandomGen, randomR)

randomDraw :: RandomGen g => Int -> Int -> g -> ([Int], g)
randomDraw n m gen = (take (max 0 (min n m)) shuffled, gen')
  where
    (shuffled, gen') = shuffle [1 .. max 0 m] gen

shuffle :: RandomGen g => [a] -> g -> ([a], g)
shuffle [] g = ([], g)
shuffle xs g = go (length xs) xs g
  where
    go 0 acc g = (acc, g)
    go len acc g =
      let (i, g') = randomR (0, len - 1) g
          swapped = swap i (len - 1) acc
      in go (len - 1) swapped g'

swap :: Int -> Int -> [a] -> [a]
swap i j xs
  | i == j    = xs
  | otherwise =
      [if k == i then xj else if k == j then xi else x
      | (k, x) <- zip [0..] xs]
  where
    xi = xs !! i
    xj = xs !! j
```

Fisher-Yates 洗牌后取前 N 个，适合 M 不大且需要多次抽样的场景。

## 方法对比

| 方法 | 代码量 | 依赖 | 特点 |
|------|--------|------|------|
| randomSelect 代理 | 1 行 | P23 | 复用已有代码 |
| 直接抽取 | ≈10 行 | 无 | 算法自包含 |
| 洗牌 + 取前 N | ≈12 行 | 无 | M 较小时效率最高 |

## 测试

```haskell
>>> import Data.List (nub)
>>> import System.Random (mkStdGen)
>>> let (xs, _) = randomDraw 6 49 (mkStdGen 111)
>>> length xs == 6
True
>>> length (nub xs) == 6 && all (`elem` [1..49]) xs
True
>>> length . fst $ randomDraw 20 5 (mkStdGen 1)   -- N > M 时只返回 5 个
5
```
