## 表示与搜索策略

棋盘是 $9\times9$ 整数列表，0 表示空格。每轮选择候选数字最少的空格（MRV），逐个尝试；候选为空立即回溯。

## 实现

### 方法一：经典回溯

```haskell
import Data.List (minimumBy, nub, sort, transpose)
import Data.Ord (comparing)

sudoku :: [[Int]] -> Maybe [[Int]]
sudoku board
  | not (validBoard board) = Nothing
  | otherwise = solve board
  where
    solve current = case blanks current of
      [] -> Just current
      positions -> firstJust
        [solve (setCell current position value) | value <- options]
        where
          position = minimumBy (comparing (length . candidates current)) positions
          options = candidates current position

blanks board = [(r,c) | r <- [0..8], c <- [0..8], board !! r !! c == 0]

candidates board (r,c) =
  [n | n <- [1..9], n `notElem` used]
  where
    row = board !! r
    column = transpose board !! c
    box = [board !! i !! j | i <- boxRange r, j <- boxRange c]
    used = row ++ column ++ box
    boxRange i = let start = i `div` 3 * 3 in [start..start+2]

setCell board (r,c) value =
  take r board ++ [take c row ++ [value] ++ drop (c+1) row] ++ drop (r+1) board
  where row = board !! r

validBoard board =
  length board == 9 && all ((== 9) . length) board
  && all (all (`elem` [0..9])) board
  && all noDuplicates (board ++ transpose board ++ boxes board)
  where
    noDuplicates xs = let values = filter (/= 0) xs
                      in length values == length (nub values)
    boxes b = [[b !! r !! c | r <- [br..br+2], c <- [bc..bc+2]]
              | br <- [0,3,6], bc <- [0,3,6]]

firstJust [] = Nothing
firstJust (Just value:_) = Just value
firstJust (Nothing:rest) = firstJust rest
```

`firstJust` 返回候选列表中的第一个成功分支。

### 方法二：约束传播

```haskell
sudokuWithPropagation :: [[Int]] -> Maybe [[Int]]
sudokuWithPropagation board
  | not (validBoard board) = Nothing
  | otherwise = solve board
  where
    solve current = do
      reduced <- propagate current
      case blanks reduced of
        [] -> Just reduced
        positions -> firstJust
          [solve (setCell reduced position value) | value <- options]
          where
            position = minimumBy (comparing (length . candidates reduced)) positions
            options = candidates reduced position

    propagate current
      | not (validBoard current) = Nothing
      | any (null . candidates current) emptyCells = Nothing
      | null forced = Just current
      | otherwise = propagate (foldl placeForced current forced)
      where
        emptyCells = blanks current
        forced =
          [(position, head options)
          | position <- emptyCells
          , let options = candidates current position
          , length options == 1]
        placeForced grid (position, value) = setCell grid position value
```

`propagate` 反复填入只有一个候选数字的格子，直到没有新的确定值。若某个空格没有候选，或同时填入后产生冲突，就立即回溯；只有传播无法继续时才选择候选最少的位置分支搜索。

## 方法对比

| 方法 | 每轮动作 | 特点 |
|------|----------|------|
| 经典回溯 | 直接选择 MRV 格子分支 | 实现短，基准清晰 |
| 约束传播 + 回溯 | 先消除全部单候选 | 搜索树通常更小 |

## 测试

```haskell
>>> fmap (all (\row -> sort row == [1..9])) (sudoku sudokuPuzzle)
Just True
>>> sudoku (replicate 9 (replicate 9 1))
Nothing
>>> sudokuWithPropagation sudokuPuzzle == sudoku sudokuPuzzle
True
```

数独搜索最坏是指数级，MRV 通常会把分支数压到很低。
