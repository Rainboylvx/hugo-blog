# P50 - 霍夫曼编码

> Huffman codes
>
> 官方模块：`Problems.P50`　核心函数：`huffman`

## 题目描述

给定字符及其频率，生成最优前缀码（霍夫曼编码）。频率越高的字符编码越短。

## 函数签名

```haskell
huffman :: [(Char, Int)] -> [(Char, String)]
```

## 实现

### 方法一：列表排序构建树

```haskell
import Data.List (sortOn)

data HuffmanTree = Leaf Char | Branch HuffmanTree HuffmanTree

huffman :: [(Char, Int)] -> [(Char, String)]
huffman [] = []
huffman [(c,_)] = [(c,"0")]
huffman frequencies = codes (snd (head (build initial)))
  where
    initial = sortOn fst [(w, Leaf c) | (c,w) <- frequencies]

    build [tree] = [tree]
    build ((w,a):(v,b):rest) =
      build (sortOn fst ((w + v, Branch a b) : rest))
    build [] = []

codes :: HuffmanTree -> [(Char, String)]
codes = walk ""
  where
    walk prefix (Leaf c) = [(c, prefix)]
    walk prefix (Branch left right) =
      walk (prefix ++ "0") left ++ walk (prefix ++ "1") right
```

反复取出权重最小的两棵树合并，用 `sortOn` 维护有序列表。

### 方法二：使用优先队列

```haskell
import Data.List (insertBy, sortOn)
import Data.Ord (comparing)

huffman :: [(Char, Int)] -> [(Char, String)]
huffman [] = []
huffman [(c,_)] = [(c,"0")]
huffman frequencies = codes (buildQueue initial)
  where
    initial = sortOn fst [(w, Leaf c) | (c,w) <- frequencies]

    buildQueue [tree] = snd tree
    buildQueue ((w1,a):(w2,b):rest) =
      buildQueue (insertBy (comparing fst) (w1 + w2, Branch a b) rest)
```

`insertBy (comparing fst)` 只比较权重，不要求 `HuffmanTree` 具有 `Ord` 实例；每次合并后把新树插回正确位置。

### 方法三：codes 用差分字符串优化

```haskell
codesShowS :: HuffmanTree -> [(Char, String)]
codesShowS tree = [(c, build "") | (c, build) <- walk id tree]
  where
    walk prefix (Leaf c) = [(c, prefix)]
    walk prefix (Branch left right) =
      walk (prefix . ('0' :)) left
      ++ walk (prefix . ('1' :)) right
```

`ShowS` 就是 `String -> String`。追加一位编码变成函数组合，只有到叶子时才把构造器应用到空串，避免在每层复制前缀。

## 测试

```haskell
>>> let table = huffman [('a',45),('b',13),('c',12),('d',16),('e',9),('f',5)]
>>> all (\(_,code) -> not (null code)) table
True
>>> huffman [('a',2),('b',1)]
[('b',"0"),('a',"1")]
>>> codesShowS (Branch (Leaf 'a') (Leaf 'b'))
[('a',"0"),('b',"1")]
```
