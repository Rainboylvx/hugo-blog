# P86 - 图着色

> Color a graph with the Welsh-Powell algorithm
>
> 官方模块：`Problems.P86`　核心函数：`colorGraph`

## 算法

Welsh-Powell 是贪心着色：按度数从高到低处理顶点，每个顶点选择尚未被邻居使用的最小正整数颜色。它保证合法，但不保证颜色数全局最少。

## 实现

### 方法一：Welsh-Powell 贪心

```haskell
import Data.List (sortOn)
import Data.Ord (Down(..))

colorGraph :: G -> [(Vertex, Int)]
colorGraph graph = Map.toAscList (foldl color Map.empty order)
  where
    vs = Set.toList (fst (canonical graph))
    order = sortOn (Down . Set.size . (`neighbors` graph)) vs

    color assigned vertex = Map.insert vertex available assigned
      where
        used = Set.fromList
          [c | neighbor <- Set.toList (neighbors vertex graph)
             , Just c <- [Map.lookup neighbor assigned]]
        available = head [c | c <- [1..], Set.notMember c used]
```

### 方法二：回溯求最少颜色

```haskell
colorGraphOptimal :: G -> [(Vertex, Int)]
colorGraphOptimal graph = Map.toAscList $ head
  [coloring
  | colorCount <- [0..length vertices]
  , Just coloring <- [assign colorCount vertices Map.empty]
  ]
  where
    vertices = sortOn
      (Down . Set.size . (`neighbors` graph))
      (Set.toList (fst (canonical graph)))

    assign _ [] coloring = Just coloring
    assign colorCount (vertex:rest) coloring = firstJust
      [assign colorCount rest (Map.insert vertex color coloring)
      | color <- [1..colorCount]
      , all (\neighbor -> Map.lookup neighbor coloring /= Just color)
            (Set.toList (neighbors vertex graph))]

    firstJust [] = Nothing
    firstJust (Just value:_) = Just value
    firstJust (Nothing:rest) = firstJust rest
```

从 0、1、2……种颜色依次尝试，找到的第一个完整着色必然使用最少颜色。高度数顶点优先可以提前制造冲突，但最坏情况仍是指数搜索。

## 方法对比

| 方法 | 保证 | 代价 |
|------|------|------|
| Welsh-Powell | 快速得到合法着色 | 不保证颜色数最少 |
| 回溯 | 得到最少颜色着色 | 指数级，只适合小图 |

## 测试

```haskell
>>> colorGraph (toG (Paths [[1,2,3,1]]))
[(1,1),(2,2),(3,3)]
>>> let graph = toG (Paths [[1,2,3,4],[1,4]])
>>> let coloring = Map.fromList (colorGraph graph)
>>> all (\(u,v) -> coloring Map.! u /= coloring Map.! v)
...     (Set.toList (snd (canonical graph)))
True
>>> maximum (map snd (colorGraphOptimal (toG (Paths [[1,2,3,1]]))))
3
```

在简单列表和集合实现下，复杂度约 $O(n^2)$。
