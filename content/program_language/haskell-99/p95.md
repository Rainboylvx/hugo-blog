# P95 - 英文数字单词

> English number words
>
> 官方模块：`Problems.P95`　核心函数：`fullWords`

## 题目描述

把非负整数逐位转换为英文单词，并用连字符连接。题目不是把 175 写成 “one hundred seventy-five”，而是 `one-seven-five`。

## 实现

```haskell
import Data.List (intercalate)

fullWords :: Integral a => a -> String
fullWords n
  | n < 0     = error "fullWords: non-negative input required"
  | n == 0    = "zero"
  | otherwise = intercalate "-" (map digitWord (digits n))

digits :: Integral a => a -> [Int]
digits 0 = []
digits n = digits (n `div` 10) ++ [fromIntegral (n `mod` 10)]

digitWord 0 = "zero"
digitWord 1 = "one"
digitWord 2 = "two"
digitWord 3 = "three"
digitWord 4 = "four"
digitWord 5 = "five"
digitWord 6 = "six"
digitWord 7 = "seven"
digitWord 8 = "eight"
digitWord 9 = "nine"
digitWord _ = error "digitWord: not a digit"
```

## 测试

```haskell
>>> fullWords 175
"one-seven-five"
>>> fullWords 0
"zero"
>>> fullWords 1002
"one-zero-zero-two"
```

`digits` 中的尾部 `++` 对位数很少的整数足够清楚；也可用累加器一次生成正序数字。
