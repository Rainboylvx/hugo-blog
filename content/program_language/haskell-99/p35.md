# P35 - 质因数分解

> Determine the prime factors of a positive integer
>
> 官方模块：`Problems.P35`　核心函数：`primeFactors`

---

[← P34 欧拉函数](./p34) | [P36 带重数的质因数分解 →](./p36)

## 题目描述

把正整数分解为按升序排列的质因数列表，重复因子要重复出现。例如 $315=3\times3\times5\times7$。

## 函数签名与实现

```haskell
primeFactors :: Integral a => a -> [a]
primeFactors n
  | n < 1     = error "primeFactors: positive input required"
  | otherwise = factor n 2
  where
    factor 1 _ = []
    factor x d
      | d * d > x      = [x]
      | x `mod` d == 0 = d : factor (x `div` d) d
      | d == 2         = factor x 3
      | otherwise      = factor x (d + 2)
```

找到因子后仍用同一个 `d` 继续除，从而保留重数。若 $d^2>x$，剩余的 $x$ 必然是质数。

## 测试

```haskell
>>> primeFactors 315
[3,3,5,7]
>>> primeFactors 13
[13]
>>> primeFactors 1
[]
```

## 复杂度

最坏情况是输入本身为素数，需要试除到平方根，时间复杂度 $O(\sqrt n)$。
