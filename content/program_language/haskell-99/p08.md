## 题目描述

去除列表中连续重复的元素，每组连续相同元素只保留一个。元素的相对顺序不变。

## 函数签名

```haskell
compress :: Eq a => [a] -> [a]
```

需要 `Eq a` 约束来比较元素是否相等。

## 思路

遍历列表，**只保留与前一个元素不同的那些元素**。即：

- 空列表 → `[]`
- 单元素 → 原样保留
- `x : y : xs`：
  - 如果 `x == y`，跳过 `y`，继续检查 `x` 和 `xs` 的头部
  - 如果 `x /= y`，保留 `x`，递归处理 `y : xs`

## 实现

### 方法一：模式匹配 + 递归

```haskell
compress :: Eq a => [a] -> [a]
compress []       = []
compress [x]      = [x]
compress (x:y:xs)
  | x == y    = compress (x:xs)
  | otherwise = x : compress (y:xs)
```

最直接的递归版本。`[x]` 模式处理单元素列表（避免不必要的对比），双元素 `(x:y:xs)` 模式决定去重还是保留。

### 方法二：使用 `map` + `group`

```haskell
import Data.List (group)

compress :: Eq a => [a] -> [a]
compress = map head . group
```

`group` 把连续重复元素打包成子列表（`"aaaabccaadeeee"` → `["aaaa","b","cc","aa","d","eeee"]`），`map head` 取每个子列表的第一个元素。

这是一个经典的 **Haskell 单行惯用法**。

### 方法三：使用 `foldr`

```haskell
compress :: Eq a => [a] -> [a]
compress = foldr go []
  where
    go x [] = [x]
    go x acc
      | x == head acc = acc
      | otherwise     = x : acc
```

从右向左 fold，如果当前元素等于已积累结果的头部，就跳过它。

## 理解 `group` 的工作方式

`Data.List.group` 是 P09 要解决的问题。但在 P08 中我们可以用它来简化实现：

```
输入: "aaaabccaadeeee"
group: ["aaaa","b","cc","aa","d","eeee"]
map head: "abcade"
```

每一组连续重复元素被压成一个单元素，恰好完成去重。

## 测试

```haskell
>>> compress "aaaabccaadeeee"
"abcade"

>>> compress [1,1,2,2,3,3,3,1,1]
[1,2,3,1]

>>> compress []
[]

>>> compress "a"
"a"
```

> 注意最后的 `1` 不会被去掉，因为 `compress` 只去**连续重复**，而这两个 `1` 被 `2,3` 隔开了。
