# P87 - 图的深度优先遍历

> Depth-first graph traversal
>
> 官方模块：`Problems.P87`　核心函数：`depthFirst`

## 题目描述

从指定顶点出发，以深度优先顺序返回所有可达顶点。邻居来自 `Set`，因此这里按升序展开，结果可复现。

## 实现

### 方法一：标准实现

```haskell
depthFirst :: G -> Vertex -> [Vertex]
depthFirst graph start = reverse (snd (visit start (Set.empty, [])))
  where
    visit vertex state@(seen, order)
      | Set.member vertex seen = state
      | otherwise = foldl (flip visit) state'
          (Set.toAscList (neighbors vertex graph))
      where state' = (Set.insert vertex seen, vertex : order)
```

状态 `(seen, order)` 显式在线程中传递；先记录当前顶点，再递归邻居，就是前序 DFS。

## 测试

```haskell
>>> let graph = toG (Paths [[1,2,3,4,5],[2,4],[6,7]])
>>> depthFirst graph 1
[1,2,3,4,5]
>>> depthFirst graph 6
[6,7]
```

每个顶点和边至多处理常数次，时间复杂度 $O(|V|+|E|)$。
