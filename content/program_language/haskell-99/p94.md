# P94 - 生成非同构正则图

> Generate regular graphs
>
> 官方模块：`Problems.P94`　核心函数：`regularGraphs`

## 题目描述

生成含 $n$ 个顶点的所有非同构 $k$-正则简单图。每个顶点度数都必须是 $k$；握手定理要求 $nk$ 为偶数。

## 实现

### 方法一：标准实现

```haskell
import Data.List (nubBy)
import qualified Data.Map.Strict as Map

regularGraphs :: Int -> Int -> [G]
regularGraphs n k
  | n < 0 || k < 0 || (n > 0 && k >= n) || odd (n*k) = []
  | otherwise = nubBy isomorphic
      [graph | chosen <- combinations edgeCount possibleEdges
             , let graph = toG (Lists (vertices, chosen))
             , all ((== k) . Set.size . (`neighbors` graph)) vertices]
  where
    vertices = [1..n]
    possibleEdges = [(u,v) | u <- vertices, v <- vertices, u < v]
    edgeCount = n * k `div` 2
```

先用握手定理固定边数，再过滤度数，最后用 P85 去除同构副本。

### 方法二：剩余度数剪枝

```haskell
regularGraphsBacktracking :: Int -> Int -> [G]
regularGraphsBacktracking n k
  | n < 0 || k < 0 || (n > 0 && k >= n) || odd (n * k) = []
  | otherwise = nubBy isomorphic
      [toG (Lists (vertices, chosen))
      | chosen <- search possibleEdges initialDegrees []]
  where
    vertices = [1..n]
    possibleEdges = [(u, v) | u <- vertices, v <- vertices, u < v]
    initialDegrees = Map.fromList [(vertex, 0) | vertex <- vertices]

    search remaining degrees chosen
      | any impossible vertices = []
      | null remaining =
          [reverse chosen | all ((== k) . degree) vertices]
      | otherwise = withoutEdge ++ withEdge
      where
        degree vertex = Map.findWithDefault 0 vertex degrees
        incident vertex (u, v) = vertex == u || vertex == v
        impossible vertex =
          degree vertex > k
          || degree vertex + length (filter (incident vertex) remaining) < k

        edge@(u, v) = head remaining
        rest = tail remaining
        withoutEdge = search rest degrees chosen
        withEdge
          | degree u >= k || degree v >= k = []
          | otherwise = search rest
              (Map.adjust (+ 1) u (Map.adjust (+ 1) v degrees))
              (edge : chosen)
```

这个版本逐条决定边是否存在，并维护每个顶点的当前度数。如果某个顶点已经超过 $k$，或者剩余相邻边全部加入也达不到 $k$，立即剪掉整棵搜索子树。

## 方法对比

| 方法 | 剪枝 |
|------|------|
| 固定边数组合 | 选满 nk/2 条边后检查所有度数 |
| 剩余度数回溯 | 每次选边后检查上限和可达下限 |

## 测试

```haskell
>>> length (regularGraphs 6 3)
2
>>> length (regularGraphs 4 2)
1
>>> regularGraphs 5 3
[]
>>> length (regularGraphsBacktracking 4 2)
1
```

这是穷举实现，候选数为 $\binom{n(n-1)/2}{nk/2}$，只适合小图。增量构边并限制剩余度数能显著提前剪枝。
