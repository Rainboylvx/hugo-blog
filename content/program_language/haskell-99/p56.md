# P56 - 判断二叉树是否对称

> Symmetric binary trees
>
> 官方模块：`Problems.P56`　核心函数：`symmetric`

## 题目描述

只比较树的结构，不比较节点值。若左子树和右子树互为镜像，则整棵树对称。

## 函数签名

```haskell
symmetric :: Tree a -> Bool
```

## 实现

### 方法一：镜像递归比较

```haskell
symmetric :: Tree a -> Bool
symmetric Empty                 = True
symmetric (Branch _ left right) = mirror left right

mirror :: Tree a -> Tree b -> Bool
mirror Empty Empty = True
mirror (Branch _ l r) (Branch _ l' r') =
  mirror l r' && mirror r l'
mirror _ _ = False
```

`mirror` 的两个类型参数可以不同，说明判定完全不关心节点值。每个节点至多访问一次，O(n)。

### 方法二：镜像路径集合

```haskell
import qualified Data.Set as Set

symmetric :: Tree a -> Bool
symmetric tree = all hasMirror paths
  where
    paths = nodePaths tree
    pathSet = Set.fromList paths
    hasMirror path = Set.member (map not path) pathSet

nodePaths :: Tree a -> [[Bool]]
nodePaths = walk []
  where
    walk _ Empty = []
    walk reversedPath (Branch _ left right) =
      reverse reversedPath
      : walk (False : reversedPath) left
      ++ walk (True : reversedPath) right
```

用 `False`、`True` 分别表示向左和向右。若每个已存在节点路径的逐位镜像也存在，整棵树的形状就对称。集合方法不读取节点值。

### 方法三：翻转后比较

```haskell
symmetric :: Tree a -> Bool
symmetric Empty = True
symmetric (Branch _ left right) = invertShape (shape left) == shape right

shape :: Tree a -> Tree ()
shape Empty = Empty
shape (Branch _ left right) = Branch () (shape left) (shape right)

invertShape :: Tree () -> Tree ()
invertShape Empty = Empty
invertShape (Branch () left right) =
  Branch () (invertShape right) (invertShape left)
```

先擦除节点值，只保留 `Tree ()` 的形状，再翻转左子树与右子树比较。因此签名不需要增加 `Eq a` 约束。

## 测试

```haskell
>>> symmetric (Branch 'x' (leaf 'a') (leaf 'b'))
True
>>> symmetric (Branch 'x' (leaf 'a') Empty)
False
>>> symmetric Empty
True
```
