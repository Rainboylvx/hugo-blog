# P56 - 判断二叉树是否对称

> Symmetric binary trees
>
> 官方模块：`Problems.P56`　核心函数：`symmetric`

## 题目描述

只比较树的结构，不比较节点值。若左子树和右子树互为镜像，则整棵树对称。

## 函数签名

```haskell
symmetric :: Tree a -> Bool
```

## 实现

### 方法一：镜像递归比较

```haskell
symmetric :: Tree a -> Bool
symmetric Empty                 = True
symmetric (Branch _ left right) = mirror left right

mirror :: Tree a -> Tree b -> Bool
mirror Empty Empty = True
mirror (Branch _ l r) (Branch _ l' r') =
  mirror l r' && mirror r l'
mirror _ _ = False
```

`mirror` 的两个类型参数可以不同，说明判定完全不关心节点值。每个节点至多访问一次，O(n)。

### 方法二：层序遍历比较

```haskell
symmetric :: Tree a -> Bool
symmetric t = levelsMatch (bfs [t])
  where
    bfs [] = []
    bfs (Empty : rest) = bfs rest
    bfs (Branch _ l r : rest) = l : r : bfs rest
```

按层从左到右收集节点，检查每层是否对称。适合对递归不熟悉的读者。

### 方法三：翻转后比较

```haskell
symmetric :: Tree a -> Bool
symmetric Empty = True
symmetric (Branch _ left right) = invert left == invert right

invert :: Tree a -> Tree a
invert Empty = Empty
invert (Branch x l r) = Branch x (invert r) (invert l)
```

将左子树翻转后与右子树比较：如果左右子树翻转后相同，则原树对称。

## 测试

```haskell
>>> symmetric (Branch 'x' (leaf 'a') (leaf 'b'))
True
>>> symmetric (Branch 'x' (leaf 'a') Empty)
False
>>> symmetric Empty
True
```
