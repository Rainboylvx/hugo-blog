# P62 - 收集二叉树指定层的节点

> Collect nodes at a given level
>
> 官方模块：`Problems.P62`　核心函数：`atLevel`

---

## 题目描述

根节点位于第 1 层。给定层号，按从左到右的顺序返回该层所有节点值。

## 实现

```haskell
atLevel :: Tree a -> Int -> [a]
atLevel Empty _ = []
atLevel _ level | level < 1 = []
atLevel (Branch x _ _) 1 = [x]
atLevel (Branch _ left right) level =
  atLevel left (level - 1) ++ atLevel right (level - 1)
```

递归下降一层时，目标层号也减一；减到 1 时当前节点就是答案。

## 测试

```haskell
>>> atLevel tree1 1
"a"
>>> atLevel tree1 3
"def"
>>> atLevel tree1 10
""
>>> atLevel tree1 0
""
```

访问深度不超过目标层，最坏时间复杂度为 $O(n)$。
