# P62 - 收集二叉树指定层的节点

> Collect nodes at a given level in a list
>
> 官方模块：`Problems.P62`　核心函数：`atLevel`

---

[← P61 收集叶子节点](./p61) | [P63 构造完全二叉树 →](./p63)

---

## 函数签名

```haskell
atLevel :: Tree a -> Int -> [a]
```

## 实现

### 方法一：递归带深度计数器

```haskell
atLevel :: Tree a -> Int -> [a]
atLevel Empty _ = []
atLevel _ level | level < 1 = []
atLevel (Branch x _ _) 1 = [x]
atLevel (Branch _ left right) level =
  atLevel left (level - 1) ++ atLevel right (level - 1)
```

递归进入左右子树时层数减 1，层数为 1 时收集节点。

### 方法二：BFS 层序遍历

```haskell
atLevel :: Tree a -> Int -> [a]
atLevel tree level = go [tree] level
  where
    go [] _ = []
    go _ 0 = []
    go nodes 1 = [x | Branch x _ _ <- nodes]
    go nodes n = go (concatMap children nodes) (n - 1)
    children Empty = []
    children (Branch _ l r) = [l, r]
```

按层展开，直到目标层。适合需要多层的场景。

## 测试

```haskell
>>> atLevel tree1 2
"bc"
>>> atLevel tree1 4
"g"
>>> atLevel tree1 5
[]
```
