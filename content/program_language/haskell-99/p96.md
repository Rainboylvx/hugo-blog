# P96 - 检查 Ada 标识符语法

> Syntax checking for Ada identifiers
>
> 官方模块：`Problems.P96`　核心函数：`isIdentifier`

## 语法规则

标识符必须以英文字母开头，之后可以包含字母、数字和下划线；下划线不能连续，也不能位于末尾。

## 实现

```haskell
isIdentifier :: String -> Bool
isIdentifier [] = False
isIdentifier (first:rest) = asciiLetter first && consume False rest
  where
    consume previousUnderscore [] = not previousUnderscore
    consume previousUnderscore (c:cs)
      | asciiLetter c || asciiDigit c = consume False cs
      | c == '_' && not previousUnderscore = consume True cs
      | otherwise = False

asciiLetter c = c `elem` ['A'..'Z'] || c `elem` ['a'..'z']
asciiDigit c  = c `elem` ['0'..'9']
```

状态 `previousUnderscore` 记录前一个字符是否为下划线；输入结束时它还为真，就说明标识符以下划线结尾。

## 测试

```haskell
>>> isIdentifier "this_is_a_long_identifier"
True
>>> isIdentifier "This_ends_in_an_underscore_"
False
>>> isIdentifier "This__has__two__underscores"
False
>>> isIdentifier "1234"
False
>>> isIdentifier "Fibonacci_sequence_1_1_2_3_5_8"
True
```

算法单次扫描字符串，时间复杂度 $O(n)$。
