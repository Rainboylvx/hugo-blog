# P64 - 二叉树布局：中序排列

> Binary tree layout: in-order

> 官方模块：`Problems.P64`　核心函数：`layoutInorder`

## 题目描述

按中序遍历顺序给二叉树节点分配坐标 $(x, y)$。$x$ 为中序遍历序号（即当前是第几个被访问的节点），$y$ 为层数。

## 函数签名

```haskell
layoutInorder :: Tree a -> Tree (a, (Int, Int))
```

## 实现

### 方法一：递归分配坐标

```haskell
layoutInorder :: Tree a -> Tree (a, (Int, Int))
layoutInorder tree = fst (layout tree 1 1)

layout :: Tree a -> Int -> Int -> (Tree (a, (Int, Int)), Int)
layout Empty next _ = (Empty, next)
layout (Branch x left right) next y =
  let (leftTree, next')   = layout left next (y + 1)
      thisNode            = Branch (x, (next', y)) leftTree Empty
      (rightTree, next'') = layout right (next' + 1) (y + 1)
  in (Branch (x, (next', y)) leftTree rightTree, next'')
```

中序遍历：先布局左子树得到 `next'`，当前节点 x = `next'`，再布局右子树，起始序号为 `next' + 1`。

### 方法二：两次遍历分离

```haskell
layoutInorder :: Tree a -> Tree (a, (Int, Int))
layoutInorder tree = annotate tree 1 (inorder tree)
  where
    inorder Empty = []
    inorder (Branch x l r) = inorder l ++ x : inorder r

    annotate Empty _ _ = Empty
    annotate (Branch _ l r) y (x:xs) =
      let (leftVals, rightVals) = splitAt (treeSize l) xs
      in Branch ((x, (1, y))) (annotate l (y+1) leftVals) (annotate r (y+1) rightVals)
```

先算出中序序列，再分发给节点。

## 测试

```haskell
>>> layoutInorder (leaf 'a')
Branch ('a',(1,1)) Empty Empty
```
