# P64 - 二叉树中序布局

> Binary tree layout using in-order positions
>
> 官方模块：`Problems.P64`　核心函数：`layoutInorder`

---

[← P63 完全二叉树](./p63) | [P65 层级等距布局 →](./p65)

## 题目描述

给每个节点附加坐标 `(x,y)`：纵坐标是深度，横坐标是节点在中序遍历中的序号。

## 实现

递归函数额外返回“下一个尚未使用的横坐标”。先布局左树，得到根坐标，再从根的下一列布局右树。

```haskell
layoutInorder :: Tree a -> Tree (a, (Int, Int))
layoutInorder tree = fst (layout tree 1 1)

layout :: Tree a -> Int -> Int -> (Tree (a, (Int, Int)), Int)
layout Empty next _ = (Empty, next)
layout (Branch value left right) next depth =
  (Branch (value, (rootX, depth)) left' right', afterRight)
  where
    (left', rootX)       = layout left next (depth + 1)
    (right', afterRight) = layout right (rootX + 1) (depth + 1)
```

## 测试

```haskell
>>> layoutInorder (Branch 'a' (leaf 'b') (leaf 'c'))
Branch ('a',(2,1))
  (Branch ('b',(1,2)) Empty Empty)
  (Branch ('c',(3,2)) Empty Empty)
```

中序遍历每个节点一次，时间复杂度 $O(n)$；所有横坐标恰好是 `1..n`。
