# P16 - 每隔 N 个删除一个元素

> Drop every N'th element from a list
>
> 官方模块: `Problems.P16`
> 核心函数: `dropEvery`

---

[← P15 每个元素重复 N 次](./p15) | [P17 在 N 处切分列表 →](./p17)

---


## 题目描述

删除列表中每隔 N 个位置的元素（从 1 开始计数）。

## 函数签名

```haskell
dropEvery :: [a] -> Int -> [a]
```

## 思路

两种思路：
1. **带计数器递归**：维护一个下标计数器，每到 N 就跳过
2. **`zip` 标号过滤**：给每个元素标上位置序号，只保留序号不是 N 的倍数的

## 实现

### 方法一：递归 + 计数器

```haskell
dropEvery :: [a] -> Int -> [a]
dropEvery [] _ = []
dropEvery (x:xs) n = go x 1 xs
  where
    go x i []     = if i == n then [] else [x]
    go x i (y:ys)
      | i == n    = go y 1 ys
      | otherwise = x : go y (i + 1) ys
```

用一个计数器 `i` 跟踪当前位置，每到 N 就丢弃当前元素。

也可以把当前元素和计数器一起追踪，更简洁的写法：

```haskell
dropEvery :: [a] -> Int -> [a]
dropEvery xs n = [x | (x, i) <- zip xs [1..], i `mod` n /= 0]
```

### 方法二：`zip` + 列表推导（推荐）

```haskell
dropEvery :: [a] -> Int -> [a]
dropEvery xs n = map fst . filter (\(_, i) -> i `mod` n /= 0) $ zip xs [1..]
```

`zip xs [1..]` 给每个元素编号，`filter` 去掉序号是 n 的倍数的元素，`map fst` 取回原值。

### 方法三：`take`/`drop` 分块

```haskell
dropEvery :: [a] -> Int -> [a]
dropEvery [] _ = []
dropEvery xs n = take (n - 1) xs ++ dropEvery (drop n xs) n
```

取前 n-1 个保留，跳过第 n 个，递归处理剩余部分。这个实现最符合"间隔删除"的语义理解。

## `dropEvery` 的求值过程

以 `dropEvery "abcdefghik" 3` 为例：

```
dropEvery "abcdefghik" 3
= take 2 "abcdefghik" ++ dropEvery (drop 3 "abcdefghik") 3
= "ab" ++ dropEvery "defghik" 3
= "ab" ++ ("de" ++ dropEvery "ghik" 3)
= "ab" ++ ("de" ++ ("gh" ++ dropEvery "k" 3))
= "ab" ++ ("de" ++ ("gh" ++ "k"))
= "abdeghk"
```

## 测试

```haskell
>>> dropEvery "abcdefghik" 3
"abdeghk"

>>> dropEvery [1,2,3,4,5] 2
[1,3,5]

>>> dropEvery "a" 3
"a"

>>> dropEvery [] 3
[]
```
