# P49 - 格雷码

> Generate n-bit Gray codes
>
> 官方模块：`Problems.P49`　核心函数：`gray`

---

[← P48 N 元真值表](./p48) | [P50 Huffman 编码 →](./p50)

## 核心构造

相邻格雷码只差一个二进制位。已知 $n-1$ 位序列 `g` 后：

1. 给 `g` 的每项加前缀 `0`；
2. 反转 `g`，再给每项加前缀 `1`。

连接处也只改变最高位，因此性质保持成立。

## 实现

```haskell
gray :: Int -> [String]
gray 0 = [""]
gray n
  | n < 0     = []
  | otherwise = map ('0' :) smaller ++ map ('1' :) (reverse smaller)
  where smaller = gray (n - 1)
```

## 测试

```haskell
>>> gray 2
["00","01","11","10"]
>>> gray 3
["000","001","011","010","110","111","101","100"]
>>> length (gray 5)
32
```

生成 $2^n$ 个长度为 $n$ 的字符串，时间和输出空间均为 $O(n2^n)$。
