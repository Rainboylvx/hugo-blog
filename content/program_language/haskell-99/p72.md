# P72 - 多路树的后序遍历

> Post-order sequence of a multiway tree
>
> 官方模块：`Problems.P72`　核心函数：`postOrderSequence`

---

[← P71 内部路径长度](./p71) | [P73 S 表达式 →](./p73)

## 思路与实现

多路树的后序遍历先依次遍历所有孩子，最后访问根节点。

```haskell
postOrderSequence :: MultiwayTree a -> [a]
postOrderSequence (MultiwayTree x children) =
  concatMap postOrderSequence children ++ [x]
```

## 测试

```haskell
>>> postOrderSequence multitree5
"gfcdeba"
>>> postOrderSequence (MultiwayTree 1 [])
[1]
```

直观版本反复使用 `++`。若树非常深，可用差异列表写成：

```haskell
postOrderSequence' tree = gather tree []
  where
    gather (MultiwayTree x children) tail =
      foldr gather (x : tail) children
```

第二版始终用 `(:)` 构造结果，整体为 $O(n)$。
