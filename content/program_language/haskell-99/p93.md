## 题目描述

保持数字顺序，在相邻数字之间插入 `+ - * / =` 和必要括号，使等号两侧按有理数运算后相等。等号恰好出现一次，不能除以零。

## 表达式生成

### 方法一：递归枚举区间切分

```haskell
import Data.List (nub)
import Data.Ratio ((%))

data Expr = Expr Rational String

expressions :: [Integer] -> [Expr]
expressions [] = []
expressions [n] = [Expr (n % 1) (show n)]
expressions numbers = concat
  [combine left right
  | split <- [1 .. length numbers - 1]
  , left  <- expressions (take split numbers)
  , right <- expressions (drop split numbers)]

combine (Expr a sa) (Expr b sb) =
  [Expr (a+b) (paren sa "+" sb),
   Expr (a-b) (paren sa "-" sb),
   Expr (a*b) (paren sa "*" sb)]
  ++ [Expr (a/b) (paren sa "/" sb) | b /= 0]
  where paren l op r = "(" ++ l ++ op ++ r ++ ")"
```

## 构造等式

```haskell
arithmeticPuzzle :: [Integer] -> [String]
arithmeticPuzzle numbers = nub
  [strip l ++ " = " ++ strip r
  | split <- [1 .. length numbers - 1]
  , Expr lv l <- expressions (take split numbers)
  , Expr rv r <- expressions (drop split numbers)
  , lv == rv]
  where
    strip text
      | length text >= 2 && head text == '(' && last text == ')' = init (tail text)
      | otherwise = text
```

为了让求值语义完全明确，这个实现保留了一些非必要括号；可在展示层根据运算符优先级进一步化简，不影响求解核心。

### 方法二：区间动态规划

```haskell
import Data.Array (Array, (!), array)

arithmeticPuzzleDP :: [Integer] -> [String]
arithmeticPuzzleDP numbers = nub
  [strip leftText ++ " = " ++ strip rightText
  | split <- [1 .. length numbers - 1]
  , Expr leftValue leftText <- table ! (0, split)
  , Expr rightValue rightText <- table ! (split, length numbers)
  , leftValue == rightValue]
  where
    table :: Array (Int, Int) [Expr]
    table = array ((0, 1), (length numbers - 1, length numbers))
      [((start, end), build start end)
      | width <- [1..length numbers]
      , start <- [0..length numbers - width]
      , let end = start + width]

    build start end
      | end == start + 1 =
          [Expr (numbers !! start % 1) (show (numbers !! start))]
      | otherwise = concat
          [combine left right
          | split <- [start + 1 .. end - 1]
          , left <- table ! (start, split)
          , right <- table ! (split, end)]

    strip text
      | length text >= 2 && head text == '(' && last text == ')' =
          init (tail text)
      | otherwise = text
```

递归版本会在不同等号位置和括号结构中重复计算相同数字区间。动态规划表以半开区间 `(start,end)` 为键，每个区间的所有表达式只生成一次，再被更大区间复用。

## 方法对比

| 方法 | 子区间结果 |
|------|------------|
| 递归枚举 | 每次遇到都重新生成 |
| 区间 DP | 缓存在数组中复用 |

## 测试

```haskell
>>> not (null (arithmeticPuzzle [2,3,5,7,11]))
True
>>> arithmeticPuzzle [1]
[]
>>> not (null (arithmeticPuzzleDP [2,3,5,7,11]))
True
```

所有二叉括号结构与四种运算组合会指数增长，使用 `Rational` 则保证等值比较没有浮点误差。
