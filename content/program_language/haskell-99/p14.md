# P14 - 复制每个元素

> Duplicate elements of a list
>
> 官方模块: `Problems.P14`
> 核心函数: `dupli`

## 题目描述

将列表中的每个元素复制一份，即每个元素在结果中连续出现两次。

## 函数签名

```haskell
dupli :: [a] -> [a]
```

## 思路

遍历列表，对每个元素输出两次：`x : x : dupli xs`。

## 实现

### 方法一：递归

```haskell
dupli :: [a] -> [a]
dupli []     = []
dupli (x:xs) = x : x : dupli xs
```

最直接的递归，每个元素放两次。

### 方法二：`concatMap`

```haskell
dupli :: [a] -> [a]
dupli = concatMap (\x -> [x, x])
```

或者更简洁地：

```haskell
dupli = concatMap (replicate 2)
```

### 方法三：列表推导

```haskell
dupli xs = [x | x <- xs, _ <- [1, 2]]
```

`_ <- [1, 2]` 让每个 `x` 被取两次。

## 测试

```haskell
>>> dupli [1, 2, 3]
[1,1,2,2,3,3]

>>> dupli []
[]

>>> dupli "hi"
"hhii"
```
